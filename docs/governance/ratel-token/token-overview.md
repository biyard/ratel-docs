# RATEL Token Overview

The RATEL token powers the Ratel ecosystem and governance.

## Token Utility

### Governance Rights
- Vote on Oracle verifications
- Vote on platform decisions
- Propose changes
- Delegate voting power

### Platform Benefits
- Access premium features
- Reduced fees
- Priority support
- Exclusive content

### Staking Rewards

- Earn from platform fees
- Liquidity provision
