# DAO Basics

Ratel operates as a Decentralized Autonomous Organization (DAO), giving the community power over platform governance.

## What is a DAO?

A Decentralized Autonomous Organization where:
- Token holders vote on decisions
- Smart contracts execute outcomes
- Transparent governance processes
- Community-driven development

## Ratel DAO Structure

### Governance Token
- RATEL token for voting rights
- One token = one vote
- Earned through participation
- Can be purchased or delegated

### Proposal System
Oracle-based verification for:
- DID attribute validation
- Community attestations
- Multi-party verification
- Consensus building

### Treasury Management
- Community-controlled funds
- Transparent accounting
- Voted allocations
- Development funding

## Participating in Governance

### Get RATEL Tokens
1. Earn through platform activity
2. Purchase on exchanges (coming soon)
3. Receive from airdrops
4. Delegate voting power

### Vote on Proposals
1. Review active Oracle verifications
2. Research the issue
3. Cast your vote
4. Track outcomes

## Governance Areas

- Platform features and roadmap
- Community guidelines
- Treasury allocation
- Partnership decisions
- Token economics
- DID verification standards

## Future Governance

Q2 2025:
- Delegation system
- Quadratic voting
- Conviction voting
- On-chain governance
