# Creating Teams

Create and configure teams for collaborative work on Ratel.

![Create Team Button](/img/screenshots/teams/create-team-button.png)
*Start creating a new team*

## Creating a Team

![Team Creation Form](/img/screenshots/teams/team-creation-form.png)
*Fill in your team details*

1. Click **Create Team** in sidebar
2. Enter team name
3. Add description
4. Upload team avatar (optional)
5. Set visibility
6. Click **Create**

## Team Settings

![Team Settings Initial](/img/screenshots/teams/team-settings-initial.png)
*Configure your team settings*

### Basic Information
- **Team Name**: Display name (required)
- **Description**: What your team does
- **Avatar**: Team profile picture
- **Website**: Optional team URL

### Visibility Options
- **Public**: Anyone can see and join
- **Private**: Invite-only
- **Hidden**: Only members can see

### Team Type
- **Organization**: Company or formal group
- **Project**: Specific project team
- **Community**: Interest-based group
- **Other**: Custom type

## Team Features

### Collaboration Tools
- Shared workspace
- Team discussions
- Document collaboration
- Task management (coming soon)

### Team Spaces
- Create team-only spaces
- Moderate discussions
- Track team activity
- Analytics dashboard

## Team Roles

Initial team creator gets:
- **Owner** role
- Full administrative access
- Ability to add admins
- Delete team permission

See [Team Permissions](./team-permissions) for detailed role information.

## Best Practices

✅ Clear team purpose in description
✅ Professional team avatar
✅ Set appropriate visibility
✅ Invite key members early

❌ Don't use vague names
❌ Avoid inappropriate content
❌ Don't create duplicate teams
❌ Don't invite spam

## After Creation

Next steps:
1. Invite team members
2. Set up team spaces
3. Configure permissions
4. Start collaborating
