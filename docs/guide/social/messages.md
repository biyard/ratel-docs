# Direct Messages

Send private messages to other Ratel users.

![Messages Inbox](/img/screenshots/social/messages-inbox.png)
*Your message inbox showing recent conversations*

## Sending Messages

![New Message Button](/img/screenshots/social/new-message-button.png)
*Start a new conversation*

1. Click Messages icon
2. Click **New Message**
3. Search for user
4. Type your message
5. Press Enter or click Send

### Message Features
- Rich text formatting
- Images and GIFs
- Link previews
- @mentions
- Read receipts (coming soon)

## Managing Conversations

![Conversation View](/img/screenshots/social/conversation-view.png)
*Full conversation thread with formatting*

### View Conversations
- All messages in chronological order
- See when messages were sent
- Know when message was read
- Search within conversations

### Conversation Actions
- Mute conversation
- Delete conversation
- Block user
- Report conversation

## Message Notifications

Control DM notifications:
- **All Messages**: Notify for every message
- **Mentions Only**: Only when mentioned
- **Off**: No DM notifications

## Message Privacy

### Who Can Message You
Settings → Privacy → Messages:
- **Everyone**: Anyone can message
- **Connections**: Only mutual follows
- **No One**: Disable messages

### Message Filtering
- Automatically filter spam
- Move requests to separate folder
- Block unwanted senders

## Group Messages

## Best Practices

✅ Respond to messages promptly
✅ Be respectful in DMs
✅ Use clear subject lines
✅ Keep conversations on-topic

❌ Don't spam users
❌ Avoid unsolicited promotions
❌ Don't share sensitive info
