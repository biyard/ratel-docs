# Text Editor

Ratel's text editor supports rich formatting to make your posts stand out.

![Formatting Toolbar](/img/screenshots/posts/formatting-toolbar.png)
*The formatting toolbar with all available options*

## Basic Formatting

### Keyboard Shortcuts
- **Bold**: Ctrl/Cmd + B
- *Italic*: Ctrl/Cmd + I
- ~~Strikethrough~~: Ctrl/Cmd + Shift + X
- `Code`: Ctrl/Cmd + E

### Headings
```
# Heading 1
## Heading 2  
### Heading 3
```

### Lists
```
- Bullet point
- Another bullet

1. Numbered item
2. Another number
```

### Quotes
```
> This is a quote
> Multiple lines work too
```

## Advanced Features

### Mentions
Type `@` to mention users:
- @username triggers notification
- User gets notified of mention
- Creates clickable link to profile

### Hashtags
Use `#` for topics:
- #technology #web3 #discussion
- Makes posts discoverable
- Creates topic pages

### Links
Paste URLs for auto-preview:
- YouTube videos embed
- Twitter posts embed
- Images show preview

### Code Blocks
Use triple backticks:
```
```javascript
function hello() {
  console.log("Hello Ratel!");
}
```
```
