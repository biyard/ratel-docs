# Media and Attachments

Enhance your posts with images, videos, and other media.

## Images

### Uploading Images

![Media Upload Button](/img/screenshots/posts/media-upload-button.png)
*Click the media button to add images to your post*

1. Click image icon in post editor
2. Select files or drag and drop
3. Images upload automatically
4. Add captions (optional)

![Image Upload Interface](/img/screenshots/posts/image-upload-interface.png)
*Drag and drop or browse to upload images*

![Multiple Images Preview](/img/screenshots/posts/multiple-images-preview.png)
*Preview and arrange multiple images before posting*

### Image Guidelines
- **Supported formats**: JPG, PNG, GIF, WebP
- **Max file size**: 10MB per image
- **Max images**: 10 per post
- **Recommended**: 1200x675px for previews

### Image Editing
- Crop and rotate before uploading
- Add alt text for accessibility
- Rearrange image order

## Link Previews

Paste any URL for rich previews:
- Article headlines and images
- YouTube video thumbnails
- Twitter post embeds
- Website meta information

Previews generate automatically when you paste links.
