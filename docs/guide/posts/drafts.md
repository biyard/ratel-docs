# Drafts

Save unfinished posts and return to them later.

## Creating Drafts

### Auto-Save
- Drafts auto-save every 30 seconds
- Resume on any device
- Saved in cloud, not locally

### Manual Save
1. Write your post
2. Click **Save Draft**
3. Access from Profile → **Drafts**

## Managing Drafts

### View Drafts
- Profile menu → **Drafts**
- Shows all saved drafts
- Ordered by last edited

![Drafts List](/img/screenshots/posts/drafts-list.png)
*Your saved drafts ready to edit or publish*

### Edit Drafts
1. Click any draft to open
2. Continue editing
3. Post or save again

### Delete Drafts
- Click **⋮** menu on draft
- Select **Delete**
- Confirm deletion

## Draft Limits

- Max 50 drafts per account
- Auto-deleted after 90 days of inactivity
- Export drafts before deletion

## Tips

✅ Use drafts for long posts
✅ Save research before posting
✅ Draft multiple posts at once
✅ Review and refine later
