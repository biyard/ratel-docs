# Notification Settings

Control how and when Ratel notifies you about activity.

![Notification Settings Overview](/img/screenshots/account/notification-settings-overview.png)
*Customize all your notification preferences in one place*

## Notification Types

### Social Notifications
- New followers
- Mentions in posts
- Replies to your comments
- Likes and reactions

### Space Notifications
- New posts in your spaces
- Space invitations
- Polls and surveys
- Deliberation events

### System Notifications
- Account security alerts
- Platform updates
- Maintenance notices

## Notification Channels

### In-App Notifications
- Real-time updates in the notification bell
- Desktop push notifications (requires permission)
- Mobile push notifications

![Notification Center](/img/screenshots/account/notification-center.png)
*Your notification center showing recent activity*

### Email Notifications
- Instant emails for important activity
- Daily digest of activity
- Weekly summary
- Custom email preferences

## Managing Notifications

### Configure Notifications
1. Settings → **Notifications**
2. Choose notification types
3. Select delivery channels
4. Set quiet hours

### Mute Specific Spaces
- Click space → **Settings** → **Mute**
- Choose: All notifications, or mentions only
- Temporary (1 day, 1 week) or permanent

### Notification Frequency
- **Real-time**: Instant notifications
- **Batched**: Hourly or daily summaries
- **Weekly**: Sunday summary email

## Do Not Disturb

Set quiet hours to avoid notifications:

1. Settings → **Notifications** → **Quiet Hours**
2. Set start and end times
3. Choose days of the week
4. Emergency notifications still come through

## Troubleshooting

### Not Receiving Notifications?
- Check notification settings are enabled
- Verify email address is correct
- Check spam/junk folder
- Ensure browser allows notifications

### Too Many Notifications?
- Reduce notification types
- Switch to daily digest
- Mute busy spaces
- Set quiet hours
