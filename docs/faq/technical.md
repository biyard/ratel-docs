# Technical Questions

Technical troubleshooting and requirements.

## System Requirements

### Browser Support
- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+

### Internet Connection
- Minimum: 1 Mbps
- Recommended: 5+ Mbps

## Common Issues

### Page Won't Load
- Clear browser cache
- Disable browser extensions
- Try incognito mode
- Check internet connection

### Can't Upload Images
- Check file size (max 10MB)
- Supported formats: JPG, PNG, GIF, WebP
- Try different browser
- Check internet speed

### Wallet Won't Connect
- Update MetaMask
- Switch to supported network
- Clear wallet cache
- Try different wallet

## Performance

### Slow Loading
- Close unused tabs
- Clear browser cache
- Disable heavy extensions
- Check internet speed
