# General Questions

Frequently asked questions about Ratel.

## What is Ratel?

Ratel is a Web3 social platform for deliberative decision-making and community governance, featuring:
- Deliberation spaces for structured discussions
- Oracle-based DID verification
- Fine-Art (d.AGIT) marketplace for artwork trading
- DAO governance with RATEL tokens

## Is Ratel free to use?

Yes! Core features are free:
- Create an account
- Join and create spaces
- Participate in deliberations
- Post and engage

Premium features available for subscription.

## What makes Ratel different?

- **Structured deliberation**: Pre-survey → Discussion → Post-survey
- **Community verification**: Oracle system for DID attributes
- **Web3 integration**: DAO governance and token economy
- **Trusted art trading**: d.AGIT platform with provenance verification

## Do I need crypto to use Ratel?

No. You can use Ratel with just an email account. Crypto features are optional.

## Is Ratel available on mobile?

Web app works on mobile browsers. Native apps coming Q3 2025.
