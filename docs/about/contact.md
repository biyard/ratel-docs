## Reporting Issues

### Bug Reports

**Platform Bugs:**
1. Check [GitHub Issues](https://github.com/biyard/ratel/issues)
2. Create detailed bug report
3. Include steps to reproduce
4. Add screenshots if applicable

**Security Vulnerabilities:**
1. Email security@biyard.co
2. Include detailed description
3. Do not publicly disclose
4. Allow time for patch

### Content Issues

**Inappropriate Content:**
1. Use in-app reporting feature
2. Or email: moderation@biyard.co
3. Include post/user URL
4. Describe violation

See also: [Community Guidelines](/legal/guidelines) • [Reporting](/security/reporting)

### Scams & Fraud

**Suspected Scams:**
1. Email: security@biyard.co
2. Include evidence
3. User/content details
4. Do not engage with scammer

:::warning Never Share
Ratel team will NEVER ask for:
- Private keys or seed phrases
- Passwords
- Payment for support
- Personal financial information
:::
