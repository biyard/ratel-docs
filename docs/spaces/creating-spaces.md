# Creating a Space

Create your own deliberation Space to facilitate policy discussions.

![Create Space Button](/img/screenshots/spaces/create-space-button.png)
*Start creating your own Space*

## Before You Create

### Plan Your Space
- Define the topic/purpose clearly
- Identify your target audience
- Decide on governance rules
- Consider verification requirements

### Requirements
- Verified email address
- Minimum account age (7 days)
- Good standing in community
- Oracle verification (for some types)

## Creation Process

### Step 1: Basic Information

![Space Creation Wizard Step 1](/img/screenshots/spaces/space-creation-wizard-step1.png)
*Enter your Space details*

1. Click **Create Space** in sidebar
2. Enter Space name
3. Write clear description
4. Add Space category/tags
5. Upload Space avatar (optional)

### Step 2: Space Settings

![Space Creation Wizard Step 2](/img/screenshots/spaces/space-creation-wizard-step2.png)
*Configure privacy and access settings*

Configure:
- **Privacy**: Public, Private, or Hidden
- **Join Requirements**: Open, Approval, Invitation
- **Verification**: Required credentials
- **Content**: Allowed post types

### Step 3: Deliberation Setup

![Space Creation Wizard Step 3](/img/screenshots/spaces/space-creation-wizard-step3.png)
*Set up deliberation structure*

Set up:
- **Pre-survey**: Initial opinion collection
- **Discussion Period**: Length of deliberation
- **Post-survey**: Final opinion collection
- **Voting Method**: How decisions are made

## Space Types

### Policy Discussion
- Focused on regulation
- Evidence-based
- Multi-stage deliberation
- DID verification optional

### Community Forum
- General discussions
- Less structured
- Open participation
- Minimal requirements

### Expert Panel
- Verified experts only
- High credential requirements
- In-depth analysis
- Oracle verification required

### Vote and Decide
- Quick decisions
- Simple voting
- Time-limited
- Clear outcomes

## Best Practices

### Space Name
✅ Clear and descriptive
✅ Relevant to topic
✅ Easy to remember
✅ Professional

❌ Vague or misleading
❌ Too long
❌ Inappropriate
❌ Duplicate of existing

### Description
✅ Explain purpose clearly
✅ Set expectations
✅ List rules
✅ Include keywords

### Initial Setup
✅ Complete all settings
✅ Add welcome post
✅ Invite initial members
✅ Set moderation rules

## After Creation

Immediate next steps:
1. Review all settings
2. Create welcome post
3. Invite members or make public
4. Start first deliberation
5. Monitor early activity

### First 24 Hours
- Engage with early joiners
- Respond to questions
- Share in other platforms
- Set the tone

### First Week
- Build core community
- Establish patterns
- Gather feedback
- Refine rules

## Space Promotion

### Within Ratel
- Share in your network
- Post in relevant spaces
- Use trending hashtags
- Engage authentically

### External
- Social media sharing
- Email invitations
- Partnership outreach
- Content marketing

## Common Mistakes

❌ Unclear purpose
❌ No moderation plan
❌ Ignoring early members
❌ Too many rules initially
❌ No engagement from creator
