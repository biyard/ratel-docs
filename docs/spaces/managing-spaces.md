## Member Management

![Member Management](/img/screenshots/spaces/member-management.png)
*Manage Space members and permissions*

### Approving Members
For approval-required Spaces:
1. Review join requests
2. Check member profiles
3. Approve or decline
4. Send welcome message

### Member Roles
- **Admin**: Full management access
- **Moderator**: Content moderation
- **Member**: Standard participant
- **Guest**: Limited access

### Member Actions
- Promote/demote roles
- Remove members
- Ban users
- Warn members

## Content Moderation

### Moderation Queue
Review:
- Reported posts
- Flagged comments
- Suspicious activity
- Spam content

### Moderation Actions
- **Approve**: Allow content
- **Remove**: Delete content
- **Warn**: Issue warning
- **Ban**: Remove and ban user

### Moderation Tools
- Auto-filters for spam
- Keyword blacklist
- Report thresholds
- Moderation logs

### Health Indicators
✅ Active participation
✅ Quality discussions
✅ Low conflict
✅ Growing membership

⚠️ Warning Signs

- Rising conflicts
- High churn rate

### Improvement Actions

## Common Challenges

### Low Engagement
Solutions:
- Post regularly
- Ask questions
- Run polls
- Start deliberations

### Conflicts
Solutions:
- Clear guidelines
- Quick moderation
- Mediation tools
- Cooling periods

### Spam
Solutions:
- Auto-filters
- Verification requirements
- Report system
- Quick action

## Best Practices

✅ Active creator presence
✅ Consistent moderation
✅ Clear rules
✅ Community recognition

❌ Absent creator
❌ Inconsistent rules
❌ Ignoring reports
❌ Favoring members
