# Managing Subscriptions

How to subscribe and manage your membership.

## Subscribe

Coming Q2 2025:

1. Settings → **Membership**
2. Choose tier
3. Enter payment details
4. Confirm subscription

## Payment Methods

- Credit/debit cards
- Cryptocurrency (RATEL tokens)
- PayPal
- Stripe

## Cancel Subscription

1. Settings → **Membership**
2. Click **Cancel Subscription**
3. Confirm cancellation
4. Access until period ends

## Change Tier

1. Settings → **Membership**
2. Click **Change Plan**
3. Select new tier
4. Prorated billing adjustment
