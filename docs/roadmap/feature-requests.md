# Feature Requests

Help shape Ratel's future by suggesting features.

## Submit a Request

1. Visit GitHub: github.com/biyard/ratel/discussions
2. Search existing requests
3. Create new discussion
4. Describe your idea
5. Community votes

## What Makes a Good Request

✅ Clear problem statement
✅ Specific solution
✅ Use cases
✅ Benefits to community
✅ Implementation considerations

## Request Process

1. **Submission**: Post on GitHub
2. **Community Vote**: Others upvote
3. **Team Review**: We evaluate
4. **Roadmap**: Add if approved
5. **Development**: Build feature
6. **Release**: Ship to users

## Popular Requests

Current top requests:
- Native mobile apps
- Video posts
- Advanced search
- Custom themes
- API webhooks

## Voting

Vote on existing requests:
- GitHub reactions
- Community polls
- Token-weighted voting (coming soon)
