# RATEL Token Airdrops

Rewarding early supporters and active community members.

## Airdrop Program

RATEL tokens distributed to:
- Early adopters (before Q2 2025)
- Active community members
- Governance participants
- Space creators

Check [Eligibility](./eligibility) for criteria.

2. Check eligibility
3. Claim tokens
4. Pay gas fees
