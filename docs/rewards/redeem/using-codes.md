# Using Reward Codes

Redeem special codes for rewards.

## Where to Find Codes

- Email campaigns
- Social media contests
- Partner promotions
- Community events
- Conference giveaways

## Redeem a Code

1. Settings → **Rewards**
2. Click "Redeem Code"
3. Enter code
4. Click "Redeem"
5. Receive reward

## Code Types

- **One-time**: Single use
- **Multi-use**: Limited quantity
- **Time-limited**: Expires after date
- **User-limited**: One per account

## Troubleshooting

**Code doesn't work?**
- Check for typos
- Verify not expired
- Ensure not already used
- Check eligibility requirements
