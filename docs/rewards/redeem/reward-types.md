# Reward Types

Different types of rewards in the Ratel ecosystem.

## Points

- Earn through activity
- Redeem for features
- Non-transferable
- Expire after 1 year

## RATEL Tokens

- Governance rights
- Premium features
- Staking rewards
- Tradeable on exchanges

## Badges

- Achievement recognition
- Profile display
- Collector's items
- Some unlock features

## NFTs

Coming soon:
- Special event NFTs
- Commemorative collectibles
- Access passes
- Fine-Art (d.AGIT) pieces
