# Earning Rewards

Earn rewards for participating in the Ratel community.

## Reward Types

- **RATEL Tokens**: Platform governance token
- **Points**: Redeemable for features
- **Badges**: Show achievements
- **NFTs**: Exclusive collectibles

## How to Earn

### Daily Activities
- Log in daily: 10 points
- Create quality post: 50 points
- Engage in discussions: 25 points
- Complete surveys: 100 points

### Achievements
- First post: 100 points
- 100 followers: 500 points
- Create space: 1000 points
- Verified DID: 2000 points

### Deliberation
- Participate: 200 points
- Complete pre & post surveys: +100 points
- High-quality contribution: +500 points

### Oracle Verification
- Vote on verifications: 50 tokens
- Accurate votes: +50 tokens bonus
- Become trusted verifier: Ongoing rewards

## Redeem Rewards

Use points for:
- Premium features
- Custom badges
- Event tickets
- Profile customization

Coming soon: RATEL token redemption
