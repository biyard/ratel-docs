# d.AGIT에 오신 것을 환영합니다

**d.AGIT**(Digital Assets Guardianship & Interchange Terminal)은 갤러리, 작가, 수집가를 위한 중저가 미술품 P2P 거래 플랫폼입니다. 블록체인 기술과 AI를 활용하여 미술품 거래의 신뢰성을 보장하고, 실물과 디지털을 연결하는 혁신적인 솔루션을 제공합니다.

![d.AGIT 플랫폼](/img/dagit-platform.png)

## d.AGIT이란?

d.AGIT은 5천만원 이하 중저가 미술품 시장의 신뢰 문제를 해결합니다. 전통적인 미술품 거래에서 발생하는 진위성 검증, 출처 추적, 가격 책정의 어려움을 혁신적인 기술로 해결합니다.

### 핵심 문제점

- **신뢰 부족**: P2P 거래에서 진품 여부를 확인하기 어려움
- **제한된 2차 시장**: 중저가 미술품을 위한 플랫폼 부재
- **출처 추적의 어려움**: 미술품의 이력을 투명하게 확인할 방법이 없음
- **불투명한 가격**: 적정 가격을 판단할 객관적 기준 부족

### d.AGIT의 솔루션

1. **Digital Art Twin (디지털 아트 트윈)**
   - 실물 미술품과 NFT를 1:1로 페어링
   - QR 코드와 시리얼 넘버로 실물 인증
   - 블록체인 기반 소유권 증명
   - 실물-디지털 소유권 동기화

2. **Oracle Pool 검증**
   - 다중 오라클의 합의 기반 진품 검증
   - 갤러리, 작가, 전문가의 독립적 인증
   - 투명한 검증 이력 블록체인 기록
   - 평판 시스템으로 오라클 품질 관리

3. **EAR (Electronic Art Records)**
   - 미술품의 모든 이력 통합 관리
   - 전시, 거래, 감정, 소유권 변경 기록
   - 타임스탬프 기반 출처 추적
   - API 연동으로 외부 시스템 통합

4. **AI 기반 가격 평가**
   - Vector DB와 RAG 기술 활용
   - 유사 작품 거래 이력 분석
   - 시장 트렌드 반영 가격 예측
   - 신뢰도 점수와 함께 가격 범위 제공

## 주요 기능

### 미술품 관리

#### 실물 미술품 등록
- 작품 정보 및 고해상도 이미지 업로드
- QR 코드 및 시리얼 넘버 자동 발급
- 작가, 갤러리 정보 연결
- 크기, 재료, 제작 연도 등 메타데이터 관리

#### NFT 발행 및 페어링
- 실물 미술품과 연동된 NFT 발행
- 블록체인 기반 소유권 인증서
- 메타데이터 S3/IPFS 저장
- 자동 페어링 및 검증

#### 미술품 이력 관리
- 전시 이력 기록
- 거래 내역 추적
- 감정 기록 저장
- 소유권 변경 이력

### 진품 검증 시스템

#### 검증 요청
```
1. 미술품 소유자가 진품 검증 요청
2. 오라클 풀에 검증 요청 전파
3. 최소 5명의 오라클 참여 필요
4. 투표 마감일까지 검증 진행
```

#### 오라클 투표
```
1. 선정된 오라클이 미술품 검토
2. 승인/거절/보류 중 투표
3. 신뢰도 점수와 코멘트 제출
4. 증거 자료(이미지, 문서) 첨부
```

#### 합의 도출
```
1. 과반(Quorum) 이상 승인 시 진품 인증
2. 결과를 블록체인에 영구 기록
3. 진품 인증서 NFT 발급
4. EAR에 검증 이력 자동 추가
```

### AI 가격 평가

#### 평가 요청
- 미술품 메타데이터 자동 분석
- 작가, 장르, 크기, 재료 기반 임베딩 생성
- 유사 작품 자동 검색

#### 가격 분석
- OpenSearch에서 유사 작품 10개 검색
- Kendra로 거래 이력 조회
- AWS Bedrock (Claude)으로 종합 분석
- 최소/최대/평균 가격 및 신뢰도 제공

### 마켓플레이스

#### 미술품 등록
- AI 추천 가격 참고
- 상세 설명 및 이미지 등록
- 검증 상태 자동 표시
- EAR 문서 자동 첨부

#### 구매 프로세스
```
1. 구매자가 작품 발견 및 검토
2. 구매 신청 및 스마트 컨트랙트 에스크로
3. 판매자 배송 및 추적 정보 제공
4. 구매자 수령 확인
5. 에스크로 자동 릴리스
6. 블록체인에서 소유권 자동 이전
7. EAR에 거래 이력 기록
```

## d.AGIT 작동 방식

### 1. 갤러리/작가의 미술품 등록

**실물 미술품 등록**:
```
1. d.AGIT 플랫폼에 로그인
2. "미술품 등록" 클릭
3. 작품 정보 입력:
   - 제목, 작가명, 카테고리
   - 크기, 재료, 제작 연도
   - 고해상도 이미지 업로드
   - 작품 설명 및 스토리
4. QR 코드 및 시리얼 넘버 자동 발급
5. 실물 작품에 QR 코드 부착
```

**NFT 발행**:
```
1. 등록된 실물 미술품 선택
2. "NFT 발행" 클릭
3. 컨트랙트 주소 선택
4. 메타데이터 자동 생성 및 업로드
5. 블록체인에 NFT 발행
6. 실물과 자동 페어링
7. 페어링 상태 확인
```

### 2. 진품 검증 받기

**검증 요청 생성**:
```
1. 미술품 상세 페이지에서 "진품 검증 요청" 클릭
2. 검증 범위 선택:
   - 진위성 확인
   - 작가 확인
   - 상태 평가
3. 추가 정보 및 증거 자료 제출
4. 오라클 풀에 요청 전파
```

**검증 진행 추적**:
```
1. 검증 요청 상태 실시간 확인
2. 참여 오라클 목록 확인
3. 투표 진행 상황 모니터링
4. 오라클 코멘트 확인
5. 합의 결과 알림 수신
```

### 3. 오라클 참여하기

**오라클 등록**:
```
1. 프로필에서 "오라클 신청" 선택
2. 전문 분야 입력 (회화, 조각 등)
3. 자격 증명 제출:
   - 감정사 자격증 (선택)
   - 소속 기관
   - 경력 증빙
4. ByFactory DAO 승인 대기
5. 승인 시 오라클 활동 시작
```

**검증 투표**:
```
1. 대시보드에서 검증 요청 확인
2. 미술품 정보 및 증거 자료 검토
3. 독립적 검증 수행
4. 투표 제출:
   - 승인/거절/보류 선택
   - 신뢰도 점수 입력 (0-100)
   - 상세 코멘트 작성
   - 증거 자료 첨부
5. 블록체인에 투표 기록
```

### 4. 마켓플레이스에서 거래하기

**판매자 프로세스**:
```
1. "마켓플레이스에 등록" 클릭
2. AI 가격 평가 요청 (선택)
3. 판매 가격 설정
4. 상세 설명 작성
5. 검증 상태 및 EAR 자동 표시
6. 등록 완료
```

**구매자 프로세스**:
```
1. 카테고리, 작가, 가격으로 작품 탐색
2. 작품 상세 정보 확인:
   - 실물 및 디지털 이미지
   - 진품 검증 이력
   - EAR 전체 기록
   - 가격 추세
3. 구매 신청
4. 스마트 컨트랙트 에스크로 결제
5. 배송 추적
6. 수령 확인
7. 소유권 자동 이전
```

## 기술 기반

d.AGIT은 최첨단 기술 스택으로 구축되었습니다:

### 서버리스 아키텍처
- **AWS Lambda**: 마이크로서비스 비즈니스 로직
- **API Gateway**: RESTful API 엔드포인트
- **DynamoDB**: Single Table Design 데이터 저장소
- **S3**: 디지털 자산 및 메타데이터 저장

### AI & 분석
- **AWS Bedrock (Claude)**: 생성형 AI 가격 분석
- **Amazon Kendra**: 지능형 검색 엔진
- **OpenSearch Vector DB**: 임베딩 벡터 유사도 검색
- **SageMaker**: 커스텀 ML 모델

### 블록체인
- **IC-OS Blockchain**: NFT 및 스마트 컨트랙트
- **NFT Canister**: EIP-721 호환 NFT 관리
- **Oracle Canister**: 오라클 풀 및 투표 관리
- **zk-DRM Canister**: 동형암호 기반 DRM

### 데이터 분석
- **AWS Glue**: ETL 파이프라인
- **Amazon Redshift**: 데이터 웨어하우스
- **Amazon Athena**: S3 기반 쿼리
- **QuickSight**: BI 대시보드

## 왜 d.AGIT인가?

**갤러리와 작가를 위해**:
- 실물과 디지털 소유권의 완벽한 연동
- 블록체인 기반 진품 인증
- 투명한 거래 이력 관리
- 글로벌 2차 시장 접근

**수집가를 위해**:
- 신뢰할 수 있는 진품 검증
- 투명한 출처 및 이력 확인
- AI 기반 적정 가격 정보
- 안전한 P2P 거래

**미술 시장을 위해**:
- 5천만원 이하 시장의 93% 활성화
- P2P 거래의 신뢰 장벽 제거
- 데이터 기반 시장 분석 제공
- 투명한 생태계 구축

수천 명의 사용자와 함께 d.AGIT에서 더 신뢰할 수 있는 미술 시장을 만들어가세요.
