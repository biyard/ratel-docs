# 빠른 시작 가이드

d.AGIT 플랫폼을 처음 사용하시나요? 이 가이드는 역할별로 5분 안에 시작할 수 있도록 도와드립니다.

## 역할 선택하기

d.AGIT에서 귀하의 역할을 선택하세요:

- **[갤러리/작가](#갤러리작가-빠른-시작)**: 미술품을 등록하고 판매하고 싶어요
- **[수집가](#수집가-빠른-시작)**: 미술품을 구매하고 싶어요
- **[오라클](#오라클-빠른-시작)**: 진품 검증 전문가로 참여하고 싶어요

---

## 갤러리/작가 빠른 시작

### 1단계: 계정 생성 및 프로필 설정

```
1. d.AGIT 홈페이지 접속
2. "가입하기" 클릭
3. 이메일 또는 소셜 로그인
4. 역할 선택: "갤러리" 또는 "작가"
5. 프로필 정보 입력:
   - 갤러리명/작가명
   - 프로필 이미지
   - 소개
   - 전문 분야
6. 지갑 연결 (MetaMask 또는 Internet Identity)
```

### 2단계: 첫 미술품 등록하기

```
1. 대시보드에서 "미술품 등록" 클릭
2. 기본 정보 입력:
   - 작품명: 예) "봄날의 풍경"
   - 작가: 본인 또는 작가 선택
   - 카테고리: 회화/조각/사진 등
3. 상세 정보:
   - 크기: 가로 × 세로 × 깊이 (cm)
   - 재료: 캔버스, 유화 등
   - 제작 연도: 2024
4. 이미지 업로드:
   - 정면 이미지 (필수)
   - 측면, 뒷면, 서명 부분 (선택)
   - 최소 1920×1080 해상도 권장
5. 작품 설명 및 스토리 작성
6. "등록" 클릭
```

**결과**: QR 코드와 시리얼 넘버가 자동 발급됩니다!

:::tip QR 코드 활용하기
발급받은 QR 코드를 출력하여 실물 미술품에 부착하세요. 구매자가 QR 코드를 스캔하면 작품의 모든 정보를 즉시 확인할 수 있습니다.
:::

### 3단계: NFT 발행 및 페어링

```
1. 등록한 미술품 상세 페이지 이동
2. "NFT 발행" 버튼 클릭
3. 컨트랙트 주소 확인 (기본값 사용 권장)
4. 메타데이터 미리보기 확인
5. "발행" 클릭 및 거래 승인
6. 블록체인 트랜잭션 완료 대기 (약 30초)
7. NFT ID 및 페어링 상태 확인
```

**결과**: 실물 미술품과 NFT가 1:1로 페어링됩니다!

### 4단계: 진품 검증 요청 (선택)

```
1. 미술품 상세 페이지에서 "진품 검증 요청" 클릭
2. 검증 범위 선택
3. 추가 증거 자료 업로드 (선택)
4. 오라클 풀에 요청 제출
5. 검증 진행 상황 모니터링
```

**소요 시간**: 평균 3-7일

### 5단계: 마켓플레이스에 등록

```
1. 검증 완료 후 "마켓플레이스에 등록" 클릭
2. AI 가격 평가 요청 (권장)
3. 판매 가격 설정
4. 배송 정보 입력
5. "등록" 클릭
```

**축하합니다!** 첫 미술품이 마켓플레이스에 등록되었습니다! 🎉

## 수집가 빠른 시작

### 1단계: 계정 생성

```
1. d.AGIT 홈페이지 접속
2. "가입하기" 클릭
3. 이메일 또는 소셜 로그인
4. 역할 선택: "수집가"
5. 프로필 정보 입력
6. 지갑 연결 (결제용)
```

### 2단계: 미술품 탐색하기

```
1. 상단 메뉴에서 "마켓플레이스" 클릭
2. 관심 카테고리 선택:
   - 회화
   - 조각
   - 사진
   - 드로잉
   - 판화
3. 필터 적용:
   - 가격 범위
   - 작가
   - 크기
   - 검증 상태
4. 정렬 방식 선택:
   - 최신순
   - 가격 낮은순/높은순
   - 인기순
```

### 3단계: 작품 상세 정보 확인

```
1. 관심 작품 클릭
2. 확인할 정보:
   - ✓ 고해상도 이미지 (확대/축소 가능)
   - ✓ 진품 검증 상태
   - ✓ EAR (Electronic Art Records)
     * 전시 이력
     * 거래 이력
     * 감정 기록
     * 소유권 변경 이력
   - ✓ 가격 분석
     * AI 예측 가격 범위
     * 유사 작품 거래 가격
     * 시장 트렌드
   - ✓ Digital Art Twin 정보
     * NFT ID
     * 컨트랙트 주소
     * 페어링 상태
```

### 4단계: 구매하기

```
1. "구매하기" 버튼 클릭
2. 배송 주소 입력
3. 결제 방법 선택:
   - 암호화폐 (ETH, ICP)
   - 신용카드 (추후 지원)
4. 스마트 컨트랙트 에스크로 확인
5. 결제 승인
6. 구매 완료!
```

### 5단계: 배송 추적 및 수령

```
1. 대시보드에서 "구매 내역" 확인
2. 실시간 배송 추적
3. 미술품 수령
4. 상태 확인:
   - 실물과 사진 일치 여부
   - 손상 여부
   - QR 코드 스캔으로 진품 확인
5. "수령 확인" 버튼 클릭
```

**결과**: 에스크로에서 판매자에게 자동 결제되고, NFT가 귀하의 지갑으로 이전됩니다!

:::tip 구매 팁
- 진품 검증이 완료된 작품을 우선 확인하세요
- EAR에서 전시 이력과 감정 기록을 확인하세요
- AI 가격 평가와 실제 판매 가격을 비교하세요
- 오라클 투표 결과와 코멘트를 읽어보세요
:::

## 오라클 빠른 시작

### 1단계: 오라클 자격 확인

다음 중 하나에 해당하시나요?

- ✓ 미술 감정사 자격증 보유
- ✓ 갤러리 운영 경력 3년 이상
- ✓ 미술 관련 학위 (학사 이상)
- ✓ 미술 전문 기관 소속
- ✓ 작가 활동 5년 이상

하나라도 해당한다면 오라클로 참여하실 수 있습니다!

### 2단계: 오라클 신청

```
1. 프로필 페이지로 이동
2. "오라클 신청" 버튼 클릭
3. 전문 분야 선택:
   - 회화 (동양화/서양화)
   - 조각
   - 사진
   - 판화
   - 기타
4. 자격 증명 제출:
   - 감정사 자격증 (선택)
   - 학위증명서
   - 경력증명서
   - 소속 기관 확인서
5. 자기소개 작성
6. "신청" 클릭
```

### 3단계: ByFactory DAO 승인 대기

```
1. DAO 회원들의 투표 진행 (평균 7일)
2. 과반 승인 시 오라클 자격 부여
3. 이메일 및 알림으로 승인 통지
4. 오라클 대시보드 접근 권한 획득
```

### 4단계: 첫 검증 투표

```
1. 오라클 대시보드에서 "대기 중인 검증 요청" 확인
2. 전문 분야에 맞는 요청 선택
3. 미술품 정보 검토:
   - 고해상도 이미지
   - 작가 정보
   - 출처 문서
   - 요청자 제출 증거
4. 독립적 검증 수행
5. 투표 제출:
   - 선택: 승인/거절/보류
   - 신뢰도 점수: 0-100
   - 상세 코멘트 작성
   - 증거 자료 첨부 (선택)
6. 블록체인에 투표 기록
```

### 5단계: 평판 관리

```
1. 정확한 검증으로 평판 점수 향상
2. 다른 오라클과의 합의율 확인
3. 검증 이력 공개 (투명성)
4. 커뮤니티 피드백 확인
```

**보상**: 검증 참여마다 토큰 보상 + 평판 점수 상승!

:::caution 주의사항
- 독립적으로 검증을 수행하세요 (다른 오라클과 협의 금지)
- 정확한 검증이 가장 중요합니다 (부정확한 투표는 평판 하락)
- 이해관계가 있는 작품은 투표하지 마세요
- 충분한 증거가 없다면 "보류"를 선택하세요
:::
