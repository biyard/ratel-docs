# 진품 검증 요청하기

d.AGIT 오라클 풀에 미술품 진품 검증을 요청하는 방법을 안내합니다.

## 진품 검증이란?

**진품 검증(Oracle Verification)**은 다수의 독립적인 미술 전문가(오라클)가 미술품의 진위성을 검증하고 합의를 통해 결과를 결정하는 프로세스입니다.

### 검증의 필요성

```
검증을 받으면:
✓ 구매자 신뢰도 증가
✓ 작품 가치 상승
✓ 마켓플레이스 우선 노출
✓ 블록체인에 영구 기록
✓ 2차 거래 시 유리
```

### 검증 범위

오라클은 다음 항목을 검증합니다:

```
1. 진위성 (Authenticity)
   - 작가의 실제 작품인지
   - 복제품이나 위작이 아닌지

2. 작가 확인 (Artist Verification)
   - 작가 정보 정확성
   - 서명 진위

3. 상태 (Condition)
   - 작품 보존 상태
   - 손상 여부
   - 복원 이력

4. 출처 (Provenance)
   - 소유권 이력
   - 구매 증빙
   - 전시 이력

5. 가격 적정성 (Valuation Opinion)
   - 현재 시장 가격 대비
   - 유사 작품 거래가 비교
```

## 검증 요청 전 준비

### 필수 정보

이미 등록되어 있어야 합니다:
- ✓ 실물 미술품 등록 완료
- ✓ 고해상도 이미지 업로드
- ✓ 기본 정보 입력 (작가, 크기, 재료 등)

### 권장 증거 자료

검증 확률을 높이기 위해 준비하세요:

```
출처 증빙:
- 구매 영수증
- 이전 소유자 증빙
- 갤러리 판매 증명서
- 경매 기록

작가 관련:
- 작가 서명 클로즈업
- 작가 증명서 (Certificate of Authenticity)
- 작가와의 서신

전문가 의견:
- 기존 감정서
- 복원 기록서
- 전시 도록

추가 사진:
- 작품 뒷면 (캔버스, 프레임)
- 측면 (입체감, 레이어)
- 텍스처 디테일
- 손상 부위 (있는 경우)
```

## 검증 요청 프로세스

### 1단계: 검증 요청 시작

```
1. 등록된 미술품 목록에서 작품 선택
2. 작품 상세 페이지 이동
3. "진품 검증 요청" 버튼 클릭
```

![검증 요청 버튼](/img/screenshots/verification-request-button.png)

### 2단계: 검증 범위 선택

```
기본 검증 (무료):
  - 진위성 확인
  - 작가 확인
  - 기본 상태 평가
  - 오라클 3명 배정
  - 예상 소요: 3-5일

고급 검증 (0.1 ETH):
  - 기본 검증 항목 전체
  - 상세 상태 평가
  - 출처 검토
  - 가격 적정성 의견
  - 오라클 5명 배정
  - 예상 소요: 5-7일

전문가 검증 (0.3 ETH):
  - 고급 검증 항목 전체
  - Platinum Oracle 배정
  - 과학적 분석 의견 (선택)
  - 복원 이력 검토
  - 오라클 7명 배정
  - 예상 소요: 7-10일
```

:::tip 검증 범위 선택 가이드
- **처음 검증받는 작품**: 기본 검증 권장
- **고가 작품 (1,000만원 이상)**: 고급 검증 권장
- **유명 작가 작품**: 전문가 검증 권장
- **출처가 불명확한 작품**: 고급 이상 검증 필수
:::

### 3단계: 증거 자료 업로드

```
1. "증거 자료 추가" 섹션에서 파일 업로드
2. 자료 유형 선택:
   - 구매 영수증
   - 감정서
   - 작가 증명서
   - 전시 도록
   - 기타
3. 각 파일에 설명 추가 (선택)
4. 최대 10개 파일, 각 50MB 이하
```

**지원 형식**:
- 문서: PDF, DOC, DOCX
- 이미지: JPG, PNG
- 스캔: PDF (권장)

### 4단계: 추가 정보 입력

```
작품 배경 (선택):
  예) "2023년 서울 갤러리 A에서 구매했으며,
      작가 본인이 직접 전달했습니다."

검증 요청 사유:
  - 마켓플레이스 등록 전 신뢰도 확보
  - 구매 전 진위 확인
  - 보험 가입 목적
  - 기증/기탁 목적
  - 기타

특이사항:
  예) "작품 하단에 소규모 복원 이력 있음 (2020년)"
```

### 5단계: 오라클 풀 선택

```
자동 배정 (권장):
  - 시스템이 전문 분야 기준으로 자동 선택
  - 평판 점수 높은 오라클 우선
  - 이해관계 충돌 자동 회피

수동 선택:
  - 오라클 목록에서 직접 선택 가능
  - 최소 3명, 최대 10명
  - 각 오라클의 전문 분야 및 평판 확인
```

![오라클 선택](/img/screenshots/oracle-selection.png)

:::caution 수동 선택 주의사항
- 전문 분야가 일치하는 오라클을 선택하세요
- 평판 점수 70 이상 오라클 권장
- 너무 많은 오라클 선택 시 검증 지연 가능
:::

### 6단계: 비용 확인 및 제출

```
검증 비용 요약:
├─ 검증 범위: 기본 검증
├─ 플랫폼 수수료: 무료
├─ 오라클 보상: 시스템 부담
└─ 총 비용: 0 ETH

검증 비용 요약 (고급):
├─ 검증 범위: 고급 검증
├─ 플랫폼 수수료: 0.02 ETH
├─ 오라클 보상: 0.08 ETH (5명 × 0.016)
└─ 총 비용: 0.1 ETH
```

**제출 절차**:
```
1. 비용 확인
2. "검증 요청 제출" 버튼 클릭
3. 지갑 팝업에서 거래 승인 (유료 검증의 경우)
4. 에스크로 컨트랙트에 비용 예치
5. 요청 제출 완료
```

## 추가 정보 요청 대응

오라클이 추가 정보를 요청할 수 있습니다:

```
요청 예시:
"작품 뒷면의 캔버스 상태 사진이 필요합니다.
특히 작가 서명 부분을 고해상도로 촬영해주세요."

대응 방법:
1. 알림 또는 이메일 확인
2. 검증 상세 페이지 이동
3. "추가 정보 제출" 클릭
4. 요청된 자료 업로드
5. "제출" 클릭
```

**응답 기한**: 요청 후 7일 이내 (미응답 시 검증 취소 가능)

## 검증 결과 확인

### 결과 유형

```
승인 (Approved):
  - 과반수 오라클이 진품으로 판정
  - 신뢰도 점수 60 이상
  - 블록체인에 검증 완료 기록
  - 작품에 "검증 완료" 배지 부여

거절 (Rejected):
  - 과반수 오라클이 위작 또는 의심으로 판정
  - 신뢰도 점수 40 미만
  - 마켓플레이스 등록 불가
  - 재검증 신청 가능 (추가 증거 필요)

보류 (Pending):
  - 합의에 도달하지 못함
  - 신뢰도 점수 40-60
  - 추가 검증 권장
  - 전문가 검증으로 업그레이드 가능
```

### 상세 결과 보기

```
검증 결과 요약:
├─ 최종 판정: 승인 (Approved)
├─ 신뢰도 점수: 87/100
├─ 참여 오라클: 5명
├─ 합의율: 100% (5/5 승인)
└─ 검증 완료일: 2025-01-26

오라클별 투표:
┌──────────────────────────────────────┐
│ 김미술 (Gold)                         │
│ ├─ 투표: 승인                         │
│ ├─ 신뢰도: 90/100                     │
│ ├─ 코멘트: "작가 서명과 화풍이 일치함" │
│ └─ 증거: signature_analysis.pdf      │
├──────────────────────────────────────┤
│ 이감정 (Silver)                       │
│ ├─ 투표: 승인                         │
│ ├─ 신뢰도: 85/100                     │
│ └─ 코멘트: "재료와 기법 확인됨"        │
└──────────────────────────────────────┘
```

### 검증 증명서 발급

```
1. 검증 결과 페이지에서 "증명서 발급" 클릭
2. PDF 형식으로 자동 생성:
   - 검증 ID 및 QR 코드
   - 작품 정보
   - 검증 결과
   - 오라클 투표 요약
   - 블록체인 트랜잭션 해시
   - ByFactory DAO 서명
3. 다운로드 또는 이메일 발송
```

## 검증 후 조치

### 승인된 경우

```
1. 작품에 "검증 완료" 배지 자동 부여
2. 마켓플레이스 우선 노출
3. 검증 증명서 발급 및 보관
4. NFT 메타데이터에 검증 상태 업데이트
5. 마켓플레이스 등록 가능
```

### 거절된 경우

```
1. 거절 사유 상세 확인
2. 추가 증거 수집
3. 30일 후 재검증 신청 가능
4. 또는 이의 제기 (DAO 심사)
```

### 보류된 경우

```
1. 오라클 코멘트에서 부족한 부분 확인
2. 추가 증거 자료 준비
3. 고급 또는 전문가 검증으로 업그레이드
4. 또는 30일 후 재신청
```

## 재검증 및 이의 제기

### 재검증 신청

```
재검증 가능한 경우:
- 첫 검증에서 거절 또는 보류
- 30일 경과 후
- 새로운 증거 자료 확보
- 작품 복원 완료

신청 방법:
1. 작품 상세 페이지에서 "재검증 신청" 클릭
2. 첫 검증과의 차이점 설명
3. 새로운 증거 자료 업로드
4. 검증 범위 선택 (업그레이드 권장)
5. 제출
```

### 이의 제기

```
이의 제기 가능한 경우:
- 검증 결과에 명백한 오류가 있는 경우
- 오라클의 부정행위 의심
- 절차상 문제가 있는 경우

절차:
1. "이의 제기" 메뉴 선택
2. 이의 사유 상세 작성
3. 증거 자료 제출
4. ByFactory DAO에 심사 요청
5. DAO 투표 (7일)
6. 과반 찬성 시 재검증 무료 진행
```

:::caution 이의 제기 주의
근거 없는 이의 제기는 계정에 불이익을 줄 수 있습니다. 명확한 증거가 있을 때만 제기하세요.
:::

## 검증 취소

### 취소 가능 시점

```
취소 가능:
- 오라클이 아직 투표하지 않은 경우
- 검증 진행률 30% 미만

취소 불가:
- 오라클 투표가 50% 이상 진행된 경우
- 검증이 완료된 경우
```

### 취소 절차 및 환불

```
1. 검증 상세 페이지에서 "취소 요청" 클릭
2. 취소 사유 입력
3. 환불 금액 확인:
   - 진행률 0%: 100% 환불
   - 진행률 1-30%: 70% 환불
   - 진행률 31% 이상: 환불 불가
4. "취소 확정" 클릭
5. 에스크로에서 자동 환불 (1-3일)
```


## 문제 해결

### 오라클이 배정되지 않음

**원인**:
- 전문 분야 오라클 부족
- 작품 카테고리 미선택
- 시스템 오류

**해결책**:
```
1. 작품 카테고리 정확히 설정
2. "자동 배정" 대신 "수동 선택" 사용
3. 검증 범위 변경 (고급 → 기본)
4. 문제 지속 시 support@dagit.io 문의
```

### 검증이 지연됨

**원인**:
- 오라클 응답 지연
- 추가 정보 요청 미확인
- 휴일 기간

**해결책**:
```
1. 추가 정보 요청 확인
2. 오라클에게 리마인더 발송 (7일 경과 후)
3. 14일 초과 시 자동으로 다른 오라클 배정
4. 긴급한 경우 고객지원 문의
```

### 검증 결과 불만족

**해결책**:
```
1. 오라클 코멘트 상세 확인
2. 부족한 증거 자료 파악
3. 30일 후 재검증 신청
4. 명백한 오류가 있다면 이의 제기
5. DAO 커뮤니티에서 조언 구하기
```

## FAQ

**Q: 검증 없이 NFT를 발행할 수 있나요?**

A: 네, 가능합니다. 하지만 검증을 받은 작품이 마켓플레이스에서 훨씬 유리합니다.

**Q: 검증 결과가 영구적인가요?**

A: 네, 블록체인에 영구 기록되며 변경할 수 없습니다. 단, 새로운 증거로 재검증은 가능합니다.

**Q: 오라클을 직접 선택할 수 있나요?**

A: 수동 선택 모드에서 가능하지만, 자동 배정이 더 객관적입니다.

**Q: 검증 비용은 환불되나요?**

A: 검증이 승인되면 환불되지 않습니다. 거절 시에는 50% 환불, 시스템 오류 시 100% 환불됩니다.

**Q: 여러 작품을 한 번에 검증 요청할 수 있나요?**

A: 네, 대량 검증 기능을 사용하면 한 번에 최대 50개까지 요청할 수 있으며 할인도 적용됩니다.
