# 실물 미술품 등록하기

실물 미술품을 d.AGIT 플랫폼에 등록하고 고유 QR 코드 및 시리얼 넘버를 발급받는 방법을 안내합니다.

## 등록 전 준비사항

### 필수 정보
- ✓ 작품 고해상도 이미지 (최소 1920×1080)
- ✓ 작품 기본 정보 (제목, 작가, 제작 연도)
- ✓ 크기 정보 (가로 × 세로, 단위: cm)
- ✓ 재료 정보

### 선택 사항
- 작가 서명 이미지
- 작품 측면/뒷면 사진
- 출처 문서 (구매 영수증, 감정서 등)
- 작품 스토리 및 배경

## 등록 프로세스

### 1단계: 미술품 등록 시작

```
1. d.AGIT 대시보드 로그인
2. 좌측 메뉴에서 "미술품" 클릭
3. "새 미술품 등록" 버튼 클릭
```

![미술품 등록 시작](/img/screenshots/artwork-register-start.png)

### 2단계: 기본 정보 입력

#### 작품 정보
```
제목*:
  예) "봄날의 풍경", "무제 #7"

작가*:
  - 본인 (작가 계정인 경우)
  - 다른 작가 선택 (갤러리인 경우)
  - 새 작가 추가

카테고리*:
  - 회화 (동양화/서양화)
  - 조각
  - 사진
  - 드로잉
  - 판화
  - 설치미술
  - 영상
  - 기타
```

#### 상세 정보
```
크기*:
  가로: ___ cm
  세로: ___ cm
  깊이: ___ cm (조각, 설치미술의 경우)

재료*:
  예) "캔버스에 유화", "대리석", "디지털 프린트"

제작 연도*:
  예) 2024

에디션 (판화, 사진의 경우):
  예) "1/50" (50개 중 1번째)
```

:::tip 크기 입력 팁
- 액자를 제외한 작품 자체의 크기를 입력하세요
- 가로가 긴 작품: 가로 > 세로
- 세로가 긴 작품: 세로 > 가로
:::

### 3단계: 이미지 업로드

#### 정면 이미지 (필수)
```
요구사항:
- 형식: JPG, PNG
- 최소 해상도: 1920×1080 (Full HD)
- 권장 해상도: 3840×2160 (4K)
- 최대 파일 크기: 50MB
- 작품 전체가 명확히 보여야 함
- 좋은 조명 (그림자 최소화)
- 정면에서 촬영
```

![이미지 업로드 요구사항](/img/screenshots/image-requirements.png)

#### 추가 이미지 (선택)
- 작가 서명 클로즈업
- 작품 측면 (입체감 확인)
- 작품 뒷면 (캔버스, 프레임 상태)
- 텍스처 디테일
- 설치 모습

:::caution 주의사항
- 워터마크를 추가하지 마세요 (d.AGIT이 자동으로 처리)
- 다른 작품과 함께 찍지 마세요
- 손가락이나 반사광이 없도록 주의
:::

### 4단계: 작품 설명 작성

#### 설명 필드
```markdown
# 작품에 대하여

이 작품은 봄날 제주도의 유채꽃밭을 그린 작품입니다.
따뜻한 햇살과 바람에 흔들리는 노란 꽃들의 생동감을
표현하고자 했습니다.

## 기법
유화 기법을 사용했으며, 붓터치를 살려 입체감을 강조했습니다.

## 작가의 말
자연의 아름다움을 통해 관람자에게 평온함을
전달하고 싶었습니다.
```

**작성 팁**:
- 마크다운 문법 사용 가능
- 작품의 영감, 기법, 의도 설명
- 작가의 개인적 스토리 추가
- 100자 이상 권장

### 5단계: 현재 위치 및 상태

#### 보관 위치
```
주소 또는 위치:
  예) "서울 종로구 갤러리 A", "작가 작업실"

상태:
  - 매우 좋음 (Mint Condition)
  - 좋음 (Excellent)
  - 양호 (Good)
  - 보통 (Fair)

특이사항:
  예) "프레임 교체됨 (2023년)", "소규모 복원 이력 있음"
```

### 6단계: 등록 완료

```
1. 입력 정보 최종 확인
2. "등록" 버튼 클릭
3. 처리 대기 (약 10초)
4. QR 코드 및 시리얼 넘버 자동 발급
```

## QR 코드 및 시리얼 넘버

### 발급 정보

등록이 완료되면 다음 정보가 자동 생성됩니다:

#### QR 코드
```json
{
  "artwork_id": "physical_artwork_12345",
  "serial_number": "ART-20250123-A1B2C3",
  "issued_at": 1706025600000,
  "issuer": "BIYARD_DAGIT",
  "version": "1.0"
}
```

- **용도**: 실물 작품 진위 확인
- **형식**: PNG 이미지 (300×300 픽셀)
- **유효기간**: 영구적

#### 시리얼 넘버
```
형식: ART-YYYYMMDD-XXXXXX
예시: ART-20250123-A1B2C3

구성:
- ART: d.AGIT 플랫폼 접두사
- 20250123: 발급 날짜 (2025년 1월 23일)
- A1B2C3: 고유 랜덤 코드
```

### QR 코드 활용하기

#### 출력 및 부착
```
1. 등록 완료 페이지에서 "QR 코드 다운로드" 클릭
2. PNG 파일 저장
3. 고품질 프린터로 출력 (최소 300 DPI)
4. 실물 작품 뒷면 또는 프레임에 부착
```

**권장 부착 위치**:
- 회화: 캔버스 뒷면 우측 하단
- 조각: 받침대 하단
- 액자 작품: 액자 뒷면
- 사진/판화: 여백 또는 뒷면

:::tip QR 코드 보호하기
- 투명 라벨 위에 인쇄하여 부착
- 물이나 먼지로부터 보호
- 정기적으로 스캔 테스트
:::

#### 구매자 검증 프로세스
```
1. 구매자가 스마트폰으로 QR 코드 스캔
2. d.AGIT 웹페이지로 자동 이동
3. 실물 작품 정보 즉시 확인:
   - 작품 이미지
   - 작가 정보
   - 등록 이력
   - 소유권 정보
   - 검증 상태
4. 실물과 정보 일치 여부 확인
```

## 등록 후 관리

### 정보 수정

```
1. "내 미술품" 목록에서 작품 선택
2. "편집" 버튼 클릭
3. 수정 가능 항목:
   - 작품 설명
   - 추가 이미지
   - 현재 위치
   - 상태 정보
4. "저장" 클릭
```

:::caution 수정 제한사항
다음 정보는 NFT 발행 후 수정할 수 없습니다:
- 제목
- 작가
- 크기
- 재료
- 제작 연도
:::

### 작품 삭제

```
1. 작품 상세 페이지 이동
2. "더 보기" 메뉴 클릭
3. "삭제" 선택
4. 확인 대화상자에서 "예" 클릭
```

**삭제 조건**:
- NFT가 발행되지 않은 경우만 가능
- 진품 검증이 진행 중인 경우 불가
- 마켓플레이스에 등록된 경우 먼저 내리기 필요

## 문제 해결

### 이미지 업로드 실패

**원인**:
- 파일 크기가 50MB 초과
- 지원하지 않는 형식 (GIF, BMP 등)
- 네트워크 연결 불안정

**해결책**:
```
1. 이미지 압축 도구 사용
2. JPG 또는 PNG로 변환
3. 네트워크 연결 확인 후 재시도
```

### QR 코드가 발급되지 않음

**해결책**:
```
1. 페이지 새로고침
2. "QR 코드 재발급" 버튼 클릭
3. 문제 지속 시 support@dagit.io로 문의
```

## FAQ

**Q: 한 번에 여러 작품을 등록할 수 있나요?**

A: 현재는 개별 등록만 지원합니다. 일괄 등록 기능은 추후 추가 예정입니다.

**Q: 작품 정보를 잘못 입력했어요. 어떻게 수정하나요?**

A: NFT 발행 전이라면 편집 기능으로 수정 가능합니다. NFT 발행 후에는 작품 설명, 이미지만 수정 가능합니다.

**Q: QR 코드를 분실했어요.**

A: 작품 상세 페이지에서 언제든지 재다운로드 가능합니다.
