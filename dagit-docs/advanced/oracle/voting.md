# 검증 투표하기

오라클로서 미술품의 진위성을 검증하고 투표하는 상세 가이드입니다.

## 검증 프로세스 개요

```
검증 플로우:
1. 검증 요청 수신 (자동 배정 또는 선택)
2. 미술품 정보 검토
3. 독립적 검증 수행
4. 투표 작성 및 제출
5. 합의 결과 확인
6. 보상 수령
```

**소요 시간**: 검증당 평균 2-4시간

## 검증 요청 확인하기

### 오라클 대시보드

로그인 후 오라클 대시보드에서 검증 요청을 확인합니다:

```
대기 중인 검증 요청:
┌───────────────────────────────────────┐
│ VR-12345                               │
│ 작품: "봄날의 풍경"                    │
│ 작가: 김철수                           │
│ 카테고리: 서양화 (유화)                │
│                                        │
│ 요청일: 2025-01-23                     │
│ 마감일: 2025-01-30 (7일 남음)          │
│                                        │
│ 오라클 참여: 2/5 (40%)                 │
│ ├─ 승인: 1                             │
│ ├─ 거절: 0                             │
│ └─ 보류: 1                             │
│                                        │
│ 난이도: 중급                           │
│ 보상: 50 DAGIT                         │
│                                        │
│ [상세보기] [검증하기] [거부하기]       │
└───────────────────────────────────────┘
```

### 필터 및 정렬

관심 있는 검증 요청을 빠르게 찾기:

```
필터:
- 카테고리: 회화, 조각, 사진 등
- 난이도: 초급, 중급, 고급
- 마감일: 오늘, 3일 이내, 7일 이내
- 상태: 대기, 진행중, 완료

정렬:
- 마감일 임박순
- 보상 높은순
- 최신순
- 참여율 낮은순
```

### 검증 선택 기준

```
✓ 전문 분야에 해당하는가?
  → 본인의 전문성을 발휘할 수 있는 작품 선택

✓ 충분한 시간이 있는가?
  → 마감일까지 최소 3일 이상 여유 확인

✓ 이해관계가 없는가?
  → 작가, 갤러리, 요청자와 관계 확인

✓ 필요한 정보가 충분한가?
  → 이미지 품질, 출처 문서 확인
```

:::tip 검증 선택 팁
- 처음에는 난이도 "초급" 또는 "중급"부터 시작하세요
- 전문 분야의 작품을 우선 선택하세요
- 마감일이 임박한 요청은 신중하게 선택하세요
:::

## 미술품 정보 검토

### 기본 정보

```
작품 정보:
- 제목: "봄날의 풍경"
- 작가: 김철수 (1970년생)
- 카테고리: 회화 > 서양화 > 풍경화
- 크기: 100cm × 80cm
- 재료: 캔버스에 유화
- 제작 연도: 2024년
- 에디션: 원본 (1/1)
```

### 이미지 검토

제공된 고해상도 이미지를 상세히 확인:

```
확인 항목:
□ 정면 이미지 (전체)
  - 해상도 충분한가? (최소 Full HD)
  - 조명이 적절한가?
  - 색감이 자연스러운가?

□ 작가 서명
  - 서명이 있는가?
  - 위치가 적절한가?
  - 서체가 일관적인가?

□ 측면/뒷면 (제공된 경우)
  - 캔버스/프레임 상태
  - 손상 여부
  - 제작 흔적

□ 텍스처 디테일
  - 붓터치 확인
  - 물감 두께
  - 크랙 패턴 (오래된 작품)
```

![이미지 검토 도구](/img/screenshots/image-review-tools.png)

**검토 도구**:
- 확대/축소 (최대 400%)
- 밝기/대비 조정
- 색상 분석
- 이미지 비교 (작가의 다른 작품)

### 출처 문서

제공된 문서를 검토:

```
문서 종류:
1. 구매 영수증
   - 발행 기관 확인
   - 날짜 및 금액
   - 작품 정보 일치 여부

2. 전시 이력
   - 전시회명 및 장소
   - 전시 카탈로그
   - 사진 자료

3. 감정서 (있는 경우)
   - 감정 기관 신뢰성
   - 감정 날짜
   - 감정 내용

4. 작가 확인서
   - 작가 서명
   - 작가 도장/스탬프
   - 발급 날짜
```

### 작가 정보

작가의 이력 및 작품 스타일 확인:

```
작가 검증:
1. d.AGIT 내 작가 프로필
   - 등록된 작가인가?
   - 다른 작품 확인
   - 스타일 일관성

2. 외부 정보 검색
   - 전시 이력
   - 수상 경력
   - 미술계 평판

3. 작품 스타일
   - 서명 패턴
   - 기법 특징
   - 주제/소재
```

## 독립적 검증 수행

### 진위성 판단 기준

```
1차 검증: 기본 정보
□ 작품 정보가 일관적인가?
□ 제작 연도가 합리적인가?
□ 크기/재료가 작가 스타일과 부합하는가?

2차 검증: 시각적 분석
□ 기법이 작가 특징과 일치하는가?
□ 서명이 다른 작품과 유사한가?
□ 물감/재료가 제작 연도와 맞는가?

3차 검증: 출처 확인
□ 문서가 신뢰할 만한가?
□ 전시/거래 이력이 검증 가능한가?
□ 모순되는 정보는 없는가?

4차 검증: 전문가 의견
□ 본인의 전문 지식으로 판단 가능한가?
□ 추가 전문가 의견이 필요한가?
□ 불확실한 부분이 있는가?
```

### 검증 방법

#### 스타일 분석
```
작가의 특징적 스타일 확인:
- 붓터치 패턴
- 색상 팔레트
- 구도 및 구성
- 서명 위치 및 형태
```

#### 재료 분석 (가능한 경우)
```
- 캔버스 종류 및 상태
- 물감 종류 (유화, 아크릴 등)
- 바니시 사용 여부
- 프레임 스타일
```

#### 시대 분석
```
- 제작 연도와 재료 일치성
- 크랙 패턴 (오래된 작품)
- 변색 정도
- 보존 상태
```

### 의심스러운 징후

다음과 같은 징후가 있다면 신중히 검토:

```
⚠️ 위조 의심 신호:
- 서명이 다른 작품과 현저히 다름
- 물감 종류가 제작 연도와 맞지 않음
- 출처 문서가 모호하거나 불완전
- 작가 스타일과 확연히 다름
- 인위적인 노화 흔적
- 디지털 조작 의심
```

:::caution 주의사항
확신이 서지 않는다면 "보류"를 선택하세요. 불확실한 상태에서 승인/거절하는 것보다 보류하는 것이 더 나은 선택입니다.
:::

## 투표 작성 및 제출

### 투표 폼 작성

```
1. 투표 선택 (필수)
   ○ 승인 (Approved)
   ○ 거절 (Rejected)
   ○ 보류 (Abstained)

2. 신뢰도 점수 (필수)
   슬라이더: 0 ──────●────── 100

   가이드:
   - 90-100: 매우 확신함
   - 70-89: 확신함
   - 50-69: 보통
   - 30-49: 의심스러움
   - 0-29: 매우 의심스러움

3. 코멘트 (필수, 최소 100자)
   "이 작품은 김철수 작가의 2024년 작품으로,
   작가의 특징적인 붓터치와 색상 팔레트가
   명확히 나타나 있습니다. 특히 서명이
   다른 작품들과 일치하며, 캔버스 상태도
   제작 연도와 부합합니다. 제공된 갤러리
   전시 이력도 확인 가능하여 진품으로
   판단합니다."

4. 증거 자료 첨부 (선택)
   - 작가의 다른 작품 비교 이미지
   - 서명 비교 분석
   - 외부 참고 자료 링크
   - 전문가 의견 (익명 처리)
```

![투표 작성 폼](/img/screenshots/vote-submission-form.png)

### 투표 유형별 가이드

#### 승인 (Approved)

다음 조건을 모두 만족할 때 승인:

```
✓ 작가의 작품이 확실함
✓ 제작 연도가 합리적임
✓ 재료 및 기법이 일치함
✓ 출처가 명확함
✓ 모순되는 정보 없음
✓ 신뢰도 점수 70 이상

코멘트 예시:
"작가의 작품 스타일과 일치하며, 서명 및
출처 문서가 명확합니다. 진품으로 판단합니다.
(신뢰도: 85점)"
```

#### 거절 (Rejected)

다음 조건 중 하나라도 해당하면 거절:

```
✗ 명백한 위조 징후 발견
✗ 작가 스타일과 완전히 다름
✗ 출처 문서가 허위로 판단됨
✗ 재료가 제작 연도와 맞지 않음
✗ 작가가 해당 시기에 작품 활동 안 함

코멘트 예시:
"서명이 작가의 다른 작품과 현저히 다르며,
붓터치 패턴도 일치하지 않습니다. 제공된
출처 문서의 날짜에도 모순이 있어 위조로
판단합니다. (신뢰도: 20점)"
```

#### 보류 (Abstained)

다음과 같은 경우 보류:

```
? 정보가 불충분함
? 이미지 품질이 낮음
? 전문 분야 외의 작품
? 추가 조사가 필요함
? 확신이 서지 않음

코멘트 예시:
"제공된 이미지의 해상도가 낮아 서명 및
붓터치를 정확히 판단하기 어렵습니다.
고해상도 이미지 또는 실물 확인이 필요하여
보류합니다."
```

### 제출 전 체크리스트

```
투표 제출 전 확인:
□ 모든 이미지를 상세히 검토했는가?
□ 출처 문서를 확인했는가?
□ 작가의 다른 작품과 비교했는가?
□ 독립적으로 판단했는가? (다른 오라클과 협의 안 함)
□ 신뢰도 점수가 적절한가?
□ 코멘트가 상세하고 명확한가?
□ 증거 자료를 첨부했는가? (있다면)
□ 이해관계 충돌이 없는가?
```

### 투표 제출

```
1. "투표 제출" 버튼 클릭
2. 최종 확인 팝업
   - 투표 내용 검토
   - 수정 불가 안내
3. "확인" 클릭
4. 블록체인 트랜잭션 서명
5. 제출 완료
```

:::danger 수정 불가
제출된 투표는 수정할 수 없습니다! 제출 전 반드시 다시 한 번 확인하세요.
:::

## 합의 결과 확인

### 투표 진행 상황

```
실시간 투표 현황:
진행률: ████████░░ 80% (4/5)

투표 분포:
├─ 승인: ███ 3표 (75%)
├─ 거절: █ 1표 (25%)
└─ 보류: 0표 (0%)

평균 신뢰도: 78점
```

### 합의 알고리즘

d.AGIT은 **Quorum(과반) 합의** 방식을 사용합니다:

```
합의 규칙:
1. 최소 오라클 수: 5명
2. 승인 조건: 60% 이상 승인 (3/5)
3. 거절 조건: 60% 이상 거절 (3/5)
4. 보류: 위 조건 미충족 시

가중치 적용:
- Platinum Oracle: 3.0x
- Gold Oracle: 2.0x
- Silver Oracle: 1.5x
- Bronze Oracle: 1.2x
- Novice Oracle: 1.0x

최종 점수 = Σ(투표 × 가중치) / Σ(가중치)
```

### 합의 결과

#### 승인 (Approved)
```
결과: 진품 인증 ✓

후속 조치:
1. 검증 인증서 NFT 발급
2. EAR에 검증 이력 추가
3. 작품 검증 배지 표시
4. 마켓플레이스 신뢰도 상승
```

#### 거절 (Rejected)
```
결과: 위조 판정 ✗

후속 조치:
1. 작품에 "검증 실패" 표시
2. 마켓플레이스 등록 차단
3. 요청자에게 거절 사유 통지
4. 재검증 요청 가능 (1회)
```

#### 보류 (Pending)
```
결과: 합의 미도달 ⊙

후속 조치:
1. 추가 오라클 배정 (최대 3명)
2. 고해상도 이미지 요청
3. 실물 확인 권장
4. 7일 후 재투표
```

```

- Platinum: 2.0x → 100 DAGIT
- Gold: 1.5x → 75 DAGIT
- Silver: 1.3x → 65 DAGIT
- Bronze: 1.1x → 55 DAGIT
- Novice: 1.0x → 50 DAGIT

합의 보너스:
- 다수 의견과 일치: +20%
- 신속 응답 (24시간 이내): +10%
- 상세 코멘트 (500자 이상): +10%

최종 보상 예시 (Gold Oracle):
50 × 1.5 × (1 + 0.2 + 0.1 + 0.1) = 105 DAGIT
```

```
사용처:
- DAO 투표 참여
```

```
이번 투표 영향:

- 상세 코멘트: +1점

현재 평판: 87 → 91/100
등급 유지: Gold Oracle 🥇
```

### 피드백 확인

```
커뮤니티 피드백:
- 도움 됨: 12명 👍
- 상세함: 8명 ⭐
- 전문적: 5명 💎

요청자 피드백:
"매우 상세한 분석과 명확한 근거를 제시해
주셔서 감사합니다. 검증 결과를 신뢰할 수
```
