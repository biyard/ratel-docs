# 오라클 되기

d.AGIT 플랫폼의 진품 검증 오라클이 되어 미술품 시장의 신뢰성을 높이는데 기여하세요.

## 오라클이란?

**오라클(Oracle)**은 미술품의 진위성을 독립적으로 검증하는 전문가입니다. 다중 오라클의 합의를 통해 진품 여부를 결정하고, 블록체인에 영구 기록합니다.

### 오라클의 역할

- ✓ 미술품 진위성 검증
- ✓ 작가 확인
- ✓ 상태 평가
- ✓ 출처 검토
- ✓ 가격 적정성 의견 제시

### 오라클 보상

```
검증 참여마다:
- 토큰 보상: 10-100 DAGIT (검증 난이도에 따라)
- 평판 점수 상승: +1-10 (정확도에 따라)
- DAO 거버넌스 참여권
- 커뮤니티 인정
```

## 오라클 자격 요건

### 필수 요건 (하나 이상)

1. **미술 감정사 자격증**
   - 한국미술감정평가협회 인증
   - 국제 감정사 자격 (RICS, ISA 등)

2. **갤러리 운영 경력**
   - 3년 이상 운영 경력
   - 정식 사업자 등록
   - 전시 실적 증빙

3. **미술 관련 학위**
   - 학사 이상 (미술사, 미술교육, 미술실기)
   - 대학원 재학 이상

4. **미술 전문 기관 소속**
   - 미술관, 박물관 큐레이터
   - 미술 연구소 연구원
   - 미술 관련 교수

5. **작가 활동**
   - 5년 이상 활동 경력
   - 개인전 3회 이상
   - 공모전 수상 경력

### 권장 요건

- 특정 장르 전문성 (회화, 조각 등)
- 미술 시장 거래 경험
- 작품 복원/보존 경험
- 미술사 연구 경력
- 국제 네트워크

## 신청 프로세스

### 1단계: 자격 확인

먼저 자신이 오라클 자격 요건을 충족하는지 확인하세요:

```
자가 진단:
□ 미술 감정사 자격증 보유
□ 갤러리 운영 3년 이상
□ 미술 관련 학사 이상 학위
□ 미술 기관 소속 (큐레이터, 연구원 등)
□ 작가 활동 5년 이상

→ 하나라도 체크되면 신청 가능!
```

### 2단계: 서류 준비

필요한 서류를 준비합니다:

#### 필수 서류
```
1. 신분증 사본
2. 이력서 (미술 관련 경력 중심)
3. 자격 증빙:
   - 감정사 자격증
   - 사업자 등록증 (갤러리)
   - 학위증명서
   - 재직증명서
   - 포트폴리오 (작가)
4. 자기소개서
```

#### 선택 서류
```
- 추천서 (미술계 인사)
- 논문 또는 저서
- 전시 경력 증빙
- 수상 경력 증빙
- 언론 보도 자료
```

### 3단계: 온라인 신청

```
1. d.AGIT 프로필 페이지 이동
2. "오라클 신청" 버튼 클릭
3. 신청 폼 작성:

기본 정보:
- 이름
- 이메일
- 전화번호
- 주소

전문 분야 선택 (중복 가능):
□ 동양화
□ 서양화 (유화, 아크릴 등)
□ 수채화
□ 조각
□ 사진
□ 판화
□ 드로잉
□ 설치미술
□ 영상

자격 구분:
○ 감정사 자격증
○ 갤러리 운영
○ 학위 보유
○ 전문 기관 소속
○ 작가 활동
○ 기타

4. 자격 증빙 파일 업로드 (PDF, 최대 20MB)
5. 자기소개서 작성 (500-1000자)
6. "제출" 클릭
```

### 4단계: ByFactory DAO 심사

```
DAO 심사 프로세스:
1. 신청서 접수 (자동)
2. 기본 자격 검토 (1-2일)
   └─ 부족한 서류 보완 요청
3. DAO 투표 공고 (3일)
4. DAO 회원 투표 (7일)
   - 찬성 50% 이상: 승인
   - 반대 50% 이상: 거절
   - 미달: 재심사
5. 결과 통지 (이메일/알림)
```

**투표 기준**:
- 자격 요건 충족 여부
- 전문성 수준
- 신뢰성 (과거 이력)
- 커뮤니티 기여도
- 이해관계 충돌 여부

### 5단계: 오라클 교육

승인되면 필수 교육을 이수해야 합니다:

```
교육 과정 (온라인):
1. 오라클 시스템 이해 (30분)
   - 역할과 책임
   - 검증 프로세스
   - 합의 메커니즘

2. 검증 방법론 (1시간)
   - 진위성 판단 기준
   - 증거 수집 방법
   - 투표 작성 가이드

3. 윤리 및 규정 (30분)
   - 독립성 유지
   - 이해관계 회피
   - 개인정보 보호

4. 시스템 사용법 (30분)
   - 플랫폼 기능
   - 투표 제출 방법
   - 보상 수령 방법

5. 평가 (10문제)
   - 80점 이상 합격
   - 불합격 시 재응시 가능

총 소요 시간: 약 3시간
```

### 6단계: 활동 시작

```
교육 이수 후:
1. 오라클 대시보드 접근 권한 부여
2. 전문 분야 배지 발급
3. 첫 검증 요청 배정 (옵션)
4. 커뮤니티 소개
```

## 오라클 의무사항

### 필수 준수사항

```
1. 독립성 유지:
   - 다른 오라클과 사전 협의 금지
   - 작품 관계자와의 이해관계 회피
   - 외부 압력 거부

2. 신속한 응답:
   - 요청 후 7일 이내 투표
   - 지연 시 사전 통지
   - 3회 연속 지연 시 경고

3. 상세한 근거:
   - 투표 이유 명확히 기재
   - 증거 자료 첨부
   - 전문적 의견 제시

4. 비밀 유지:
   - 검증 중인 정보 외부 유출 금지
   - 투표 결과 사전 공개 금지
   - 작품 관련 개인정보 보호

5. 지속적 활동:
   - 월 최소 1건 검증 참여
   - 3개월 미활동 시 자격 정지
   - 6개월 미활동 시 자격 박탈
```

### 금지사항

```
❌ 다른 오라클과 투표 전 협의
❌ 요청자/작가와 직접 접촉
❌ 검증 결과에 대한 보상 요구
❌ 허위 증거 제출
❌ 전문 분야 외 검증 참여
❌ 다중 계정 생성
❌ 투표 위임 또는 대리
```

## FAQ

**Q: 오라클 승인까지 얼마나 걸리나요?**

A: 서류 접수 후 DAO 투표까지 평균 10-14일 소요됩니다.

**Q: 검증 참여를 거부할 수 있나요?**

A: 네, 전문 분야가 아니거나 이해관계가 있는 경우 거부할 수 있습니다.

**Q: 오라클 활동으로 얼마나 벌 수 있나요?**

A: 활동 수준에 따라 다르지만, 월 5-10건 검증 시 약 1,500-3,500 DAGIT 토큰을 받을 수 있습니다.

**Q: 잘못된 투표를 했다면 어떻게 되나요?**

A: 합의 결과와 다른 투표는 평판 점수에 영향을 줄 수 있지만, 근거가 명확하다면 불이익은 없습니다.

**Q: 오라클을 그만두려면?**

A: 대시보드에서 "오라클 탈퇴" 신청하면 됩니다. 진행 중인 검증 완료 후 탈퇴 처리됩니다.
