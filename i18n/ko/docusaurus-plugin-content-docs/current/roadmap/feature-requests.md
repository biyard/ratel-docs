# 기능 요청

기능 제안으로 Ratel의 미래를 만들어가세요.

## 요청 제출

1. GitHub 방문: github.com/biyard/ratel/discussions
2. 기존 요청 검색
3. 새 토론 생성
4. 아이디어 설명
5. 커뮤니티 투표

## 좋은 요청이란

✅ 명확한 문제 정의
✅ 구체적인 해결책
✅ 사용 사례
✅ 커뮤니티에 대한 이점
✅ 구현 고려사항

## 요청 프로세스

1. **제출**: GitHub에 게시
2. **커뮤니티 투표**: 다른 사람들이 추천
3. **팀 검토**: 우리가 평가
4. **로드맵**: 승인 시 추가
5. **개발**: 기능 구축
6. **릴리스**: 사용자에게 배포

## 인기 요청

현재 인기 요청:
- 네이티브 모바일 앱
- 비디오 게시물
- 고급 검색
- 사용자 지정 테마
- API 웹훅

## 투표

기존 요청에 투표:
- GitHub 리액션
- 커뮤니티 설문조사
- 토큰 가중 투표(출시 예정)
