# Ratel에 오신 것을 환영합니다

Ratel은 소셜 미디어의 참여성과 의미 있는 숙의를 위한 구조화된 도구, 그리고 신뢰할 수 있는 P2P 거래를 결합한 차세대 커뮤니티 플랫폼입니다. 블록체인 기술을 기반으로 구축된 Ratel은 커뮤니티가 투명한 거버넌스를 통해 집단적 의사결정을 내리고, 전문 시장에서 신뢰할 수 있는 교환을 촉진합니다.

![Ratel 홈페이지](/img/ratel-homepage.png)

## Ratel이란?

Ratel은 두 가지 핵심 기능을 제공합니다:

### 1. 숙의 및 설문 스페이스

정책 논의, 커뮤니티 의사결정, 여론 수렴을 위한 구조화된 환경을 만듭니다. Ratel의 숙의 프레임워크는 커뮤니티가 다음을 할 수 있도록 돕습니다:

- **사전 설문 실시**: 토론 시작 전 초기 의견 수집
- **익명 토론 촉진**: 검증된 익명 참여를 통한 개방적 대화 가능
- **의견 변화 추적**: 사후 설문을 통해 숙의가 관점에 미치는 영향 측정
- **결과 분석**: 포괄적인 분석을 통한 합의 형성 이해

**최적 대상**: 정책 입안자, 커뮤니티 조직자, 교육 기관, 구조화된 의사결정이 필요한 모든 그룹.

### 2. Fine-Art (d.AGIT): 신뢰할 수 있는 미술품 거래

**d.AGIT** (Digital Assets Guardianship & Interchange Terminal)은 갤러리, 작가, 수집가 간의 신뢰 문제를 해결하는 중저가 미술품(5만원 이하 / 약 5,000달러) P2P 마켓플레이스:

- **전자 미술품 기록(EAR)**: 미술품 출처에 대한 포괄적인 디지털 문서화
- **오라클 검증**: 미술품 진위성을 위한 다자간 인증 시스템
- **블록체인 인증서**: 소유권 및 거래 이력의 불변 증명
- **디지털 아트 트윈**: 실물 미술품과 페어링된 NFT 표현

**최적 대상**: 미술품 수집가, 신진 작가, 갤러리, 합리적인 가격의 미술 시장 참여자.

## 주요 기능

### 숙의 스페이스 기능

#### 스페이스 생성 및 구성
- 팀 기반 또는 공개 숙의 스페이스 생성
- 객관식 및 주관식 형식의 맞춤형 설문 질문 설계
- 특정 주제별 토론 게시판 설정
- 인구통계 또는 자격 증명 기반 참여자 패널 구성
- 익명 또는 신원 공개 참여 모드 선택

#### 사전 숙의 설문
- 토론 전 기준선 의견 수집
- 다양한 질문 유형 지원(객관식, 순위, 자유 텍스트)
- 계층화된 분석을 위한 인구통계 데이터 수집
- 참여 요구사항 및 자격 증명 설정

#### 숙의 단계
- 주제별로 구성된 구조화된 토론 게시판
- 답글 및 반응이 있는 스레드형 대화
- 민감한 주제에 대한 익명 게시 옵션
- 멀티미디어 지원(텍스트, 이미지, 비디오, 링크)
- 실시간 참여자 활동 추적

#### 사후 숙의 설문
- 의견 변화 측정을 위한 사전 설문과 동일한 질문
- 숙의가 관점을 어떻게 바꾸었는지 분석
- 합의 영역과 남은 이견 식별
- 추가 분석을 위한 결과 내보내기

#### 분석 및 인사이트
- 사전/사후 설문 비교 대시보드
- 인구통계 그룹별 의견 변화 추적
- 토론 참여 지표
- 참여자 기여도 분석
- 연구 목적의 데이터 내보내기

### Fine-Art (d.AGIT) 플랫폼 기능

#### 신뢰 인프라
Ratel은 P2P 미술품 거래의 근본적인 과제인 독립적 당사자 간 신뢰 부족 문제를 해결합니다.

**문제점**:
- 미술품 경매 거래의 93%가 5만원 미만이지만 전체 가치의 12%만 차지
- 합리적인 가격의 미술품을 위한 제한적인 2차 시장 플랫폼
- 갤러리, 작가, 수집가 간 P2P 거래의 신뢰 문제
- 미술품 진위성 및 출처 검증의 어려움

**우리의 솔루션**:

1. **전자 미술품 기록(EAR)**
   - 포괄적인 미술품 메타데이터 및 문서
   - 창작부터 현재 소유자까지의 출처 추적
   - 고해상도 이미지 및 상태 보고서
   - 전문가 감정 및 가격 이력

2. **오라클 풀 검증**
   - 다자간 인증 시스템
   - 갤러리, 작가, 전문가 검증
   - 합의 기반 진위성 확인
   - 투명한 검증 이력

3. **디지털 아트 트윈**
   - 실물 미술품의 NFT 표현
   - 블록체인 기반 소유권 인증서
   - 변조 방지 거래 이력
   - 휴대 가능한 디지털 소유권 증명

4. **데이터 기반 가격 책정**
   - 미술 시장 데이터 수집
   - AI 기반 가격 추천
   - 과거 거래 분석
   - 공정한 시장 가치 추정

## 숙의 스페이스 작동 방식

### 1. 설정 단계(생성자)

**스페이스 생성**:
```
1. 팀 또는 조직 생성
2. 공동 생성자 및 중재자 초대
3. 다음 내용으로 숙의 게시물 초안 작성:
   - 주제 설명
   - 배경 맥락
   - 참고 자료(비디오, 문서)
```

**스페이스 구성**:
```
1. 개요 및 목표 설정
2. 사전 설문 질문 설계
3. 토론 게시판 주제 생성
4. 참여자 초대(선택적 자격 증명 요구사항 포함)
5. 인구통계 패널 설정
6. 익명 또는 신원 공개 참여 선택
7. 초대된 참여자에게 게시
```

### 2. 사전 설문 단계(참여자)

```
1. 참여자가 초대장 수신
2. 스페이스 목표 및 자료 검토
3. 사전 설문 질문 완료
4. 초기 설문 결과 보기(활성화된 경우)
5. 숙의 스페이스 입장
```

### 3. 숙의 단계

```
1. 생성자가 숙의 시작
2. 참여자가 토론에 참여:
   - 특정 주제에 대한 게시판 글 읽기
   - 게시물 및 댓글에 답글 달기
   - 아이디어를 담은 새 게시물 작성
   - 다른 사람의 기여에 반응하기
3. 익명 게시로 신원 보호하면서 책임성 유지
4. 숙의 시작 후 늦게 가입한 사람은 제한될 수 있음
```

### 4. 최종 설문 및 분석

```
1. 생성자가 최종 설문 게시(사전 설문과 동일한 질문)
2. 참여자가 최종 설문 완료
3. 시스템이 의견 변화 분석:
   - 질문별 사전/사후 비교
   - 인구통계 그룹 분석
   - 합의 식별
   - 토론 영향 측정
```

## Fine-Art (d.AGIT) 작동 방식

### 1. 미술품 등록

**갤러리 및 작가용**:
```
1. 전자 미술품 기록(EAR) 생성
   - 미술품 세부 정보(제목, 작가, 매체, 크기)
   - 고해상도 이미지
   - 출처 문서
   - 창작 및 소유권 이력

2. 오라클 검증 제출
   - 갤러리 검증
   - 작가 확인
   - 전문가 인증(필요한 경우)

3. 디지털 아트 트윈 발행
   - NFT 인증서 생성
   - 실물 미술품에 링크
   - 블록체인에 저장
```

### 2. 등록 및 발견

```
1. 마켓플레이스에 미술품 등록
   - AI 추천 기반 가격 설정
   - EAR 문서 포함
   - 검증 상태 표시

2. 구매자가 미술품 발견
   - 카테고리, 작가, 스타일별 탐색
   - 전체 출처 보기
   - 검증 이력 확인
   - 가격 추세 확인
```

### 3. 거래 프로세스

```
1. 구매자가 구매 시작
2. 스마트 계약이 에스크로로 결제 보관
3. 판매자가 추적 가능한 배송으로 미술품 발송
4. 구매자가 수령 및 상태 확인
5. 판매자에게 결제 릴리스
6. 블록체인에서 소유권 이전
7. 새 소유자로 EAR 업데이트
```

### 4. 2차 시장

```
1. 소유자가 전체 이력과 함께 재판매 가능
2. EAR이 모든 출처를 이어감
3. 오라클 검증이 유효 유지
4. 미래 구매자가 거래 이력 확인 가능
5. 가격 이력이 평가에 도움
```

## 기술 기반

Ratel은 최첨단 기술로 구축되었습니다:

- **블록체인**: 안전하고 투명한 기록 보관
- **스마트 계약**: 자동화된 무신뢰 거래
- **DAO 거버넌스**: 커뮤니티 주도 의사결정
- **마이크로 서비스 아키텍처**: 확장 가능한 모듈식 플랫폼
- **AWS 클라우드 인프라**: 안정적이고 고성능 호스팅
- **영지식 증명**: 개인정보 보호 검증(익명 숙의용)

## 왜 Ratel인가?

**커뮤니티를 위해**:
- 좋아요와 공유를 넘어 의미 있는 숙의로
- 증거 기반 집단 의사결정
- 토론이 마음을 어떻게 바꾸는지 추적
- 투명한 거버넌스 프로세스 구축

**미술 시장을 위해**:
- 5만원 미만 거래의 93%에 접근
- P2P 거래의 신뢰 장벽 제거
- 출처 및 진위성 검증 제공
- 투명한 2차 시장 창출

수천 명의 사용자와 함께 Ratel에서 더 나은 커뮤니티와 더 신뢰할 수 있는 시장을 구축하세요.
