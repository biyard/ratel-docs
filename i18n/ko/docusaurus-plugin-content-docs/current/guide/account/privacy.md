# 개인정보 설정

Ratel에서 누가 귀하의 정보를 볼 수 있고 귀하와 상호작용할 수 있는지 제어하세요.

## 프로필 개인정보

### 가시성 설정
- **공개 프로필**: 모든 사람에게 표시
- **등록된 사용자만**: 로그인한 사용자만
- **비공개 프로필**: 승인된 팔로워만

### 콘텐츠 가시성
- **게시물**: 누가 게시물을 볼 수 있는지
- **스페이스**: 누가 참여 중인 스페이스를 볼 수 있는지
- **활동**: 누가 좋아요와 댓글을 볼 수 있는지

## 통신 개인정보

### 연락할 수 있는 사람
- **모든 사람**: 누구나 메시지 보낼 수 있음
- **연결만**: 팔로우하는 사람만
- **없음**: 다이렉트 메시지 비활성화

### 알림
- 새 메시지 이메일 알림
- 멘션 푸시 알림
- 주간 다이제스트 이메일

## 데이터 개인정보

### 데이터 수집
- **분석**: Ratel 개선 지원(선택 해제 가능)
- **개인화**: 경험 사용자 지정
- **제3자 공유**: 파트너와 공유되는 데이터 제어

### 데이터 다운로드
모든 Ratel 데이터 내보내기:
1. 설정 → 개인정보 → **데이터 다운로드**
2. 48시간 이내에 이메일로 전송
3. 다운로드 링크는 7일간 유효

## 차단 및 음소거

### 사용자 차단
- 프로필이나 콘텐츠를 볼 수 없음
- 연락하거나 멘션할 수 없음
- 이전 상호작용은 귀하에게 계속 표시됨

### 사용자 음소거
- 피드에서 게시물 숨기기
- 멘션 및 직접 상호작용은 여전히 표시됨
- 음소거한 사실을 알지 못함

## 개인정보 관리

1. 프로필 사진 클릭 → **설정**
2. **개인정보**로 이동
3. 필요에 따라 설정 조정
4. 변경 사항은 즉시 적용됨
