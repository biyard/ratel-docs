# 알림 설정

Ratel이 활동에 대해 알리는 방법과 시기를 제어하세요.

## 알림 유형

### 소셜 알림
- 새 팔로워
- 게시물 멘션
- 댓글 답글
- 좋아요 및 반응

### 스페이스 알림
- 스페이스의 새 게시물
- 스페이스 초대
- 설문조사 및 여론조사
- 숙의 이벤트

### 시스템 알림
- 계정 보안 알림
- 플랫폼 업데이트
- 유지보수 공지

## 알림 채널

### 인앱 알림
- 알림 벨의 실시간 업데이트
- 데스크톱 푸시 알림(권한 필요)
- 모바일 푸시 알림

### 이메일 알림
- 중요한 활동에 대한 즉시 이메일
- 일일 활동 다이제스트
- 주간 요약
- 사용자 지정 이메일 기본 설정

## 알림 관리

### 알림 구성
1. 설정 → **알림**
2. 알림 유형 선택
3. 전달 채널 선택
4. 조용한 시간 설정

### 특정 스페이스 음소거
- 스페이스 클릭 → **설정** → **음소거**
- 선택: 모든 알림 또는 멘션만
- 임시(1일, 1주) 또는 영구

### 알림 빈도
- **실시간**: 즉시 알림
- **일괄 처리**: 시간별 또는 일일 요약
- **주간**: 일요일 요약 이메일

## 방해 금지

조용한 시간을 설정하여 알림 피하기:

1. 설정 → **알림** → **조용한 시간**
2. 시작 및 종료 시간 설정
3. 요일 선택
4. 긴급 알림은 여전히 수신됨

## 문제 해결

### 알림을 받지 못하시나요?
- 알림 설정이 활성화되어 있는지 확인
- 이메일 주소가 정확한지 확인
- 스팸/정크 폴더 확인
- 브라우저가 알림을 허용하는지 확인

### 알림이 너무 많나요?
- 알림 유형 줄이기
- 일일 다이제스트로 전환
- 바쁜 스페이스 음소거
- 조용한 시간 설정
