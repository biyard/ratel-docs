# 프로필 설정

커뮤니티에서 자신을 나타내기 위해 Ratel 프로필을 사용자 지정하세요.

## 프로필 정보

### 기본 정보
- **표시 이름**: Ratel에서 공개되는 이름
- **사용자 이름**: 고유한 @핸들
- **자기소개**: 간단한 설명(최대 160자)
- **프로필 사진**: 정사각형 이미지, 최소 400x400px 권장

### 확장 프로필
- **위치**: 도시/지역(선택 사항)
- **웹사이트**: 개인 또는 전문 웹사이트
- **소셜 링크**: Twitter, LinkedIn 등

## 프로필 편집

1. 오른쪽 상단의 프로필 사진 클릭
2. **설정** → **프로필** 선택
3. 정보 업데이트
4. **변경 사항 저장** 클릭

## 프로필 가시성

프로필 정보를 볼 수 있는 사람 제어:

- **공개**: 누구나 프로필 볼 수 있음
- **회원 전용**: 등록된 사용자만
- **비공개**: 팔로우하는 사람만

## 프로필 인증

준비 중인 기능으로 다음을 통해 신원 인증:
- 이메일 인증(현재 사용 가능)
- 소셜 계정 연결
- DID(탈중앙화 신원) 통합
- Oracle 커뮤니티 인증

## 훌륭한 프로필을 위한 팁

- 명확하고 인식 가능한 프로필 사진 사용
- 관심사를 강조하는 간결한 자기소개 작성
- 관련 소셜 링크 추가
- 정보를 최신 상태로 유지
