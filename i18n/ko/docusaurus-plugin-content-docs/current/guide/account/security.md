# 계정 보안

이러한 보안 기능 및 모범 사례로 Ratel 계정을 보호하세요.

![보안 설정 페이지](/img/screenshots/account/security-settings-page.png)
*계정을 보호하는 포괄적인 보안 제어*

## 비밀번호 보안

### 강력한 비밀번호
- 최소 8자
- 문자, 숫자 및 기호 혼합
- 일반적인 단어나 패턴 피하기
- 다른 사이트의 비밀번호 재사용하지 않기

### 비밀번호 변경
1. 설정 → **보안** → **비밀번호 변경**
2. 현재 비밀번호 입력
3. 새 비밀번호 입력 및 확인
4. **비밀번호 업데이트** 클릭

![비밀번호 변경 양식](/img/screenshots/account/password-change-form.png)
*안전한 비밀번호 변경 인터페이스로 비밀번호 업데이트*

## 활성 세션

계정에 로그인한 기기 보기 및 관리:

1. 설정 → **보안** → **활성 세션**
2. 활성 기기 및 위치 목록 보기
3. 인식되지 않는 기기에서 **로그아웃** 클릭

![활성 세션](/img/screenshots/account/active-sessions.png)
*계정에 로그인한 모든 기기 모니터링 및 관리*

## 복구 옵션

### 이메일 복구
- 이메일 주소를 최신 상태로 유지
- 비밀번호 재설정을 활성화하려면 이메일 인증
- 복구 이메일을 받지 못하면 스팸 폴더 확인
- 안전하고 접근 가능한 이메일 계정 사용

## Web3 지갑 보안

Web3 지갑을 연결하는 경우:

- 시드 구문이나 개인 키를 **절대 공유하지 마세요**
- 큰 금액에는 하드웨어 지갑 사용
- 서명하기 전에 거래 세부 정보 확인
- 사용하지 않을 때 지갑 연결 해제

## 보안 알림

다음을 감지하면 이메일로 알려드립니다:
- 새 기기 로그인
- 비밀번호 변경
- 이메일 주소 변경
- 의심스러운 활동

## 모범 사례

✅ 고유하고 강력한 비밀번호 사용
✅ 이메일 보안 유지
✅ 정기적으로 활성 세션 검토
✅ 의심스러운 링크 클릭하지 않기
✅ 공유 기기에서 로그아웃
✅ 비밀번호 관리자 사용

❌ 비밀번호 공유하지 않기
❌ VPN 없이 공용 WiFi 사용하지 않기
❌ 비밀번호 재사용하지 않기
❌ 보안 알림 무시하지 않기
❌ 공유 컴퓨터의 브라우저에 비밀번호 저장하지 않기
