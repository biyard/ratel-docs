# 다이렉트 메시지

다른 Ratel 사용자에게 비공개 메시지를 보내세요.

## 메시지 보내기

1. **메시지** 아이콘 클릭
2. **새 메시지** 클릭
3. 사용자 검색
4. 메시지 입력 및 전송

### 메시지 기능
- 텍스트 서식
- 이미지 첨부
- 링크 미리보기
- 읽음 확인
- 입력 중 표시기

## 메시지 개인정보

### 누가 메시지를 보낼 수 있는지
개인정보 설정에서 설정:
- **모든 사람**: 누구나 메시지 가능
- **팔로잉**: 팔로우하는 사람만
- **없음**: DM 비활성화

### 메시지 요청
- 팔로워가 아닌 사람의 메시지는 요청으로 이동
- 수락하여 대화 시작
- 거절하여 무시

## 대화 관리

### 대화 음소거
- 대화 열기
- **⋮** → **음소거** 클릭
- 알림 중지

### 대화 삭제
- 대화 열기
- **⋮** → **삭제** 클릭
- 받은 편지함에서만 제거

### 사용자 차단
- 모든 메시지 방지
- 온라인 상태를 볼 수 없음
- 과거 메시지는 유지

## 그룹 메시지

준비 중: 여러 사람에게 메시지 보내기
