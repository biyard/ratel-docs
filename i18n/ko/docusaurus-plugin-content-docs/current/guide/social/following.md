# 팔로잉 및 팔로워

다른 사람과 연결하고 Ratel 네트워크를 구축하세요.

## 사용자 팔로우

### 팔로우 방법
- 사용자의 프로필 방문
- **팔로우** 버튼 클릭
- 피드에 게시물 표시

### 팔로우 취소
- 사용자의 프로필 방문
- **팔로잉** → **팔로우 취소** 클릭
- 알림을 받지 않음

## 팔로워

### 누가 팔로우할 수 있는지
개인정보 설정 기반:
- **공개**: 누구나 팔로우 가능
- **승인**: 팔로워 승인 필요
- **비공개**: 초대 전용

### 팔로워 관리
1. 프로필 → **팔로워**
2. 모든 팔로워 보기
3. 팔로워 제거(비공개 게시물을 볼 수 없음)

## 팔로우 제안

팔로우할 사람 발견:
- 관심사 기반 추천
- 스페이스의 사람들
- 인기 크리에이터
- 트렌딩 주제

## 피드 알고리즘

피드 표시:
1. 팔로우하는 사람의 게시물
2. 스페이스의 게시물
3. 추천 콘텐츠(비활성화 가능)

정렬 기준:
- **최신**: 시간순
- **인기**: 가장 많은 참여
- **혼합**: 알고리즘 균형
