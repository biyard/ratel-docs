# 네트워크

Ratel에서 연결을 보고 관리하세요.

## 네트워크 개요

프로필 → **네트워크**에서 액세스:

- **팔로잉**: 팔로우하는 사람
- **팔로워**: 나를 팔로우하는 사람
- **상호**: 서로 팔로우하는 사람
- **제안**: 추천 연결

## 연결 찾기

### 다른 플랫폼에서
- Twitter, LinkedIn에서 가져오기
- 이메일로 찾기
- 프로필 링크 공유

### Ratel에서
- 사용자 이름으로 검색
- 관심사별로 탐색
- 스페이스 탐색
- 게시물 참여 확인

## 목록

준비 중: 연결을 목록으로 구성

## 네트워크 개인정보

네트워크를 볼 수 있는 사람 제어:
- 팔로잉 목록 숨기기
- 팔로워 목록 숨기기
- 상호 연결만 표시

설정 → 개인정보 → **네트워크 가시성**

## 네트워크 품질

품질 높은 네트워크 구축:
✅ 공유 관심사를 가진 사람 팔로우
✅ 콘텐츠에 참여
✅ 같은 스페이스에 참여
✅ 의미 있는 상호작용

❌ 대량 팔로우/팔로우 취소 금지
❌ 팔로우-포-팔로우 방식 피하기
❌ 연결 스팸 금지
