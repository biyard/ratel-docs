# 탐색 및 발견

Ratel에서 트렌딩 콘텐츠, 인기 스페이스 및 흥미로운 사람들을 발견하세요.

## 탐색 페이지

탐색하려면 탐색 메뉴에서 액세스:

### 트렌딩 주제
- 인기 해시태그
- 활발한 토론
- 바이럴 게시물
- 현재 이벤트

### 추천 스페이스
- 관심사에 맞는 스페이스
- 활발한 커뮤니티
- 새로운 출시
- 추천 스페이스

### 인기 크리에이터
- 활발한 커뮤니티 구성원
- 주제 전문가
- 떠오르는 스타
- 인증된 계정

## 검색

### 빠른 검색
- 검색창에 입력
- 즉시 결과
- 표시: 사람, 게시물, 스페이스

### 고급 검색
- 유형별 필터링
- 날짜 범위
- 참여 수준
- 특정 스페이스

### 검색 팁
- 정확한 구문에 따옴표 사용
- 해시태그 검색에 # 사용
- 사용자 이름 검색에 @ 사용
- 더 나은 결과를 위해 필터 결합

## 발견 알고리즘

콘텐츠 추천 기준:
- 관심사 및 활동
- 참여 중인 스페이스
- 팔로우하는 사람
- 참여 패턴

### 발견 사용자 지정
설정 → **기본 설정**:
- 관심 주제
- 콘텐츠 유형
- 언어 기본 설정
- 민감한 콘텐츠 숨기기
