# 임시 저장

완성되지 않은 게시물을 저장하고 나중에 다시 작성하세요.

## 임시 저장 작성

### 자동 저장
- 30초마다 자동 저장
- 모든 기기에서 재개
- 로컬이 아닌 클라우드에 저장

### 수동 저장
1. 게시물 작성
2. **임시 저장** 클릭
3. 프로필 → **임시 저장**에서 액세스

## 임시 저장 관리

### 임시 저장 보기
- 프로필 메뉴 → **임시 저장**
- 저장된 모든 임시 저장 표시
- 마지막 편집순으로 정렬

### 임시 저장 편집
1. 임시 저장을 클릭하여 열기
2. 편집 계속
3. 게시하거나 다시 저장

### 임시 저장 삭제
- 임시 저장의 **⋮** 메뉴 클릭
- **삭제** 선택
- 삭제 확인

## 임시 저장 제한

- 계정당 최대 50개 임시 저장
- 비활성 상태 90일 후 자동 삭제
- 삭제 전에 임시 저장 내보내기

## 팁

✅ 긴 게시물에 임시 저장 사용
✅ 게시 전에 연구 내용 저장
✅ 한 번에 여러 게시물 임시 저장
✅ 나중에 검토 및 수정
