# 게시물 작성

Ratel 커뮤니티와 생각, 질문, 아이디어를 공유하세요.

## 게시물 작성

1. **게시물 작성** 버튼 클릭(오른쪽 상단 또는 사이드바)
2. 콘텐츠 작성
3. 서식, 미디어 또는 설문조사 추가(선택 사항)
4. 대상 선택(공개, 스페이스 전용 또는 팔로워)
5. **게시** 클릭

## 게시물 유형

### 텍스트 게시물
- 최대 5,000자
- 리치 텍스트 서식
- 링크 미리보기

### 이미지 게시물
- 최대 10개 이미지 업로드
- 지원: JPG, PNG, GIF, WebP
- 이미지당 최대 10MB

### 설문조사 게시물
- 객관식 질문 추가
- 설문조사 기간 설정(1시간~7일)
- 실시간 결과 보기

### 숙의 게시물
- 구조화된 토론 형식
- 사전 설문조사 → 토론 → 사후 설문조사
- 숙의 스페이스에서 사용 가능

## 서식 옵션

- **굵게**, *기울임꼴*, ~~취소선~~
- 제목(H1, H2, H3)
- 글머리 기호 및 번호 목록
- 인용 블록
- 코드 블록
- @ 멘션
- # 해시태그

## 임시 저장

나중에 완성하기 위해 임시 저장:
1. 게시물 작성
2. **임시 저장** 클릭
3. 프로필 → **임시 저장**에서 액세스
4. 준비되면 편집 및 게시

## 모범 사례

✅ 명확하고 매력적인 제목 작성
✅ 가독성을 위해 단락 사용
✅ 관련 이미지 추가
✅ 게시 전 교정
✅ 발견을 위한 해시태그 사용

❌ 전체 대문자 피하기(소리지르기)
❌ 중복 콘텐츠 게시하지 않기
❌ 스팸이나 과도한 링크 금지
❌ 커뮤니티 가이드라인 준수
