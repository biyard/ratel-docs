# 미디어 및 첨부파일

이미지, 비디오 및 기타 미디어로 게시물을 향상시키세요.

## 이미지

### 이미지 업로드
1. 게시물 편집기에서 이미지 아이콘 클릭
2. 파일 선택 또는 드래그 앤 드롭
3. 이미지 자동 업로드
4. 캡션 추가(선택 사항)

### 이미지 가이드라인
- **지원 형식**: JPG, PNG, GIF, WebP
- **최대 파일 크기**: 이미지당 10MB
- **최대 이미지**: 게시물당 10개
- **권장**: 미리보기용 1200x675px

### 이미지 편집
- 업로드 전 자르기 및 회전
- 접근성을 위한 대체 텍스트 추가
- 이미지 순서 재정렬

## 비디오

준비 중: 기본 비디오 업로드

현재는 다음을 통해 비디오 공유:
- YouTube 링크(자동 임베드)
- Twitter 비디오 링크
- 직접 비디오 URL

## 문서

준비 중: PDF 및 문서 첨부

## GIF

- 이미지처럼 애니메이션 GIF 업로드
- 또는 내장 GIF 검색 사용
- 최대 10MB 크기

## 링크 미리보기

리치 미리보기를 위해 URL 붙여넣기:
- 기사 헤드라인 및 이미지
- YouTube 비디오 썸네일
- Twitter 게시물 임베드
- 웹사이트 메타 정보

링크를 붙여넣으면 미리보기가 자동으로 생성됩니다.
