# 스레드 및 댓글

댓글 및 스레드 토론을 통해 대화에 참여하세요.

## 게시물에 댓글 달기

1. 게시물을 클릭하여 세부 정보 보기
2. 댓글 상자로 스크롤
3. 댓글 작성
4. **답글** 클릭

### 댓글 기능
- 전체 텍스트 서식
- @멘션 및 #해시태그
- 중첩 답글(최대 5단계)
- 댓글에 좋아요 및 반응

## 스레드 구조

### 메인 게시물
- 원래 게시물이 스레드 시작
- 참여 통계 표시(좋아요, 댓글, 공유)
- 클릭하여 전체 스레드 확장

### 답글
- 메인 게시물에 직접 답글
- 게시물 아래에 시간순으로 표시
- 정렬 가능: 최신, 인기, 논란

### 중첩 답글
- 특정 댓글에 답글
- 대화 하위 스레드 생성
- 최대 5단계 깊이

## 스레드 알림

다음의 경우 알림을 받습니다:
- 누군가 게시물에 답글
- 누군가 댓글에 답글
- 누군가 스레드에서 멘션
- 게시물 작성자가 답글(댓글을 단 경우)

### 알림 관리
- 특정 스레드 음소거
- 답글 알림 끄기
- 설정 → 알림에서 조정

## 스레드 에티켓

✅ 주제 유지
✅ 답글 전에 전체 스레드 읽기
✅ 특정 요점에 답글할 때 인용
✅ 좋은 요점 인정

❌ 대화 방해하지 않기
❌ 반복적인 댓글 피하기
❌ 네크로포스트 금지(매우 오래된 스레드에 답글)
