# 팀 권한

역할 기반 권한으로 팀 구성원이 할 수 있는 작업을 제어하세요.

## 권한 시스템

### 소유자 권한
- 팀 삭제
- 소유권 이전
- 모든 설정 관리
- 모든 역할 할당

### 관리자 권한
- 구성원 초대 및 제거
- 팀 설정 수정
- 채널 생성
- 콘텐츠 관리

### 구성원 권한
- 게시 및 댓글
- 팀 콘텐츠 보기
- 토론 참여
- 팀 리소스 액세스

### 게스트 권한
- 제한된 콘텐츠 보기
- 지정된 게시물에 댓글
- 팀 설정 액세스 없음

## 사용자 지정 권한

준비 중: 세분화된 권한 제어

포함 예정:
- 사용자 지정 역할
- 채널별 권한
- 콘텐츠 중재 권한
- 통합 관리

## 채널 권한

팀 내에서 비공개 채널 생성:
1. 팀 → **채널**
2. 새 채널 만들기
3. 가시성 설정
4. 구성원 할당
