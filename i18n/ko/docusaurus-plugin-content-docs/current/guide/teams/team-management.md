# 팀 관리

팀을 효과적으로 관리하고 모든 구성원의 참여를 유지하세요.

## 구성원 추가

### 이메일로 초대
1. 팀 설정 → **구성원**
2. **초대** 클릭
3. 이메일 주소 입력
4. 초기 역할 설정
5. 초대 보내기

### 링크로 초대
1. 팀 설정 → **초대 링크**
2. 링크 복사
3. 팀과 공유
4. 만료 설정(선택 사항)

## 구성원 제거

1. 팀 설정 → **구성원**
2. 구성원 찾기
3. **⋮** → **제거** 클릭
4. 제거 확인

제거된 구성원:
- 즉시 액세스 손실
- 팀 콘텐츠 볼 수 없음
- 과거 기여는 유지

## 구성원 역할

준비 중: 사용자 지정 역할

기본 역할:
- **소유자**: 전체 제어
- **관리자**: 팀 관리
- **구성원**: 표준 액세스
- **게스트**: 제한된 액세스

## 팀 활동

팀 활동 모니터링:
- 활성 구성원
- 최근 게시물
- 참여 통계
- 성장 추세

## 모범 사례

✅ 명확한 팀 가이드라인 설정
✅ 새 구성원 환영
✅ 참여 장려
✅ 정기적인 점검
✅ 기여 인정
