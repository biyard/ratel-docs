# 팀 만들기

팀을 통해 Ratel에서 특정 그룹과 협업할 수 있습니다.

## 팀이란?

팀은 다음을 위한 비공개 협업 공간입니다:
- 작업 그룹
- 프로젝트 팀
- 스터디 그룹
- 조직

## 팀 만들기

1. 탐색에서 **팀** 클릭
2. **팀 만들기** 클릭
3. 팀 세부 정보 입력:
   - 팀 이름
   - 설명
   - 개인정보 수준
4. **만들기** 클릭

## 팀 기능

### 비공개 작업 공간
- 팀 전용 게시물 및 토론
- 공유 파일 및 리소스
- 비공개 채널
- 팀 캘린더

### 구성원 관리
- 이메일 또는 사용자 이름으로 구성원 초대
- 역할 및 권한 할당
- 구성원 활동 추적
- 구성원 제거

### 협업 도구
- 공유 임시 저장
- 팀 숙의
- 협업 문서
- 작업 관리

## 팀 설정

팀 → **설정**에서 구성:
- 팀 정보 및 브랜딩
- 구성원 권한
- 알림 기본 설정
- 통합 옵션
