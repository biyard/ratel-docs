# 중재

건강한 커뮤니티를 유지하기 위해 스페이스를 효과적으로 중재하세요.

## 중재 도구

### 콘텐츠 중재
- 게시물/댓글 제거
- 사용자 편집 또는 경고
- 콘텐츠 숨기기 또는 플래그 지정
- 반복 위반자 차단

### 사용자 관리
- 방해가 되는 사용자 음소거
- 임시 정지
- 영구 차단
- 이의제기 프로세스

### 자동화(출시 예정)
- 스팸 자동 필터링
- 키워드 차단
- 속도 제한
- 자동 중재 규칙

## 명확한 규칙 설정

### 필수 규칙
- 존중하기
- 주제에 충실하기
- 스팸 금지
- 괴롭힘 금지
- 불법 콘텐츠 금지

### 스페이스별 규칙
커뮤니티에 맞게 사용자 지정:
- 주제 제한
- 콘텐츠 형식
- 게시 빈도
- 언어 요구사항

## 중재 모범 사례

### 일관된 시행
- 규칙을 동등하게 적용
- 결정 문서화
- 명확하게 의사소통
- 실수로부터 배우기

### 팀 중재
- 신뢰할 수 있는 중재자 모집
- 책임 정의
- 정기적인 의사소통
- 중재 가이드라인

### 갈등 해결
1. 양측의 의견 경청
2. 증거 검토
3. 공정하게 규칙 적용
4. 결정 설명
5. 향후를 위해 문서화

## 일반적인 문제

### 스팸
- 자동 필터 사용
- 의심스러운 게시물 수동 검토
- 반복 위반자 차단
- 계정 나이/평판 요구

### 트롤
- 트롤에게 먹이 주지 않기
- 방해가 되는 콘텐츠 제거
- 패턴이 계속되면 차단
- 커뮤니티 보호

### 주제 이탈 게시물
- 부드러운 알림
- 적절한 스페이스로 리디렉션
- 지속되면 제거
- 주제 채널 만들기

역할 포함 예정:
- 선임 중재자
- 주제 중재자
- 봇 중재자
- 수습 중재자
