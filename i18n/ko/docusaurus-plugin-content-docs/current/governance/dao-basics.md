# DAO 기본

Ratel은 탈중앙화 자율 조직(DAO)으로 운영되어 커뮤니티에 플랫폼 거버넌스에 대한 권한을 부여합니다.

## DAO란 무엇인가요?

탈중앙화 자율 조직:
- 토큰 보유자가 의사결정에 투표
- 스마트 계약이 결과 실행
- 투명한 거버넌스 프로세스
- 커뮤니티 주도 개발

## Ratel DAO 구조

### 거버넌스 토큰
- 투표권을 위한 RATEL 토큰
- 1 토큰 = 1 투표
- 참여를 통해 획득
- 구매 또는 위임 가능

### 제안 시스템
다음을 위한 오라클 기반 검증:
- DID 속성 검증
- 커뮤니티 증명
- 다자간 검증
- 합의 구축

### 재무 관리
- 커뮤니티 통제 자금
- 투명한 회계
- 투표로 결정되는 할당
- 개발 자금 지원

## 거버넌스 참여

### RATEL 토큰 획득
1. 플랫폼 활동을 통해 획득
2. 거래소에서 구매(출시 예정)
3. 에어드랍을 통해 수령
4. 투표권 위임

### 제안에 투표
1. 활성 오라클 검증 검토
2. 이슈 조사
3. 투표 진행
4. 결과 추적

## 거버넌스 영역

- 플랫폼 기능 및 로드맵
- 커뮤니티 가이드라인
- 재무 할당
- 파트너십 결정
- 토큰 이코노믹스
- DID 검증 표준

## 향후 거버넌스

2025년 2분기:
- 위임 시스템
- 제곱 투표
- 신념 투표
- 온체인 거버넌스
