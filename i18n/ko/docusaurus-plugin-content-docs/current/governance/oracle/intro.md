# 오라클 이해하기

Ratel 플랫폼에서 커뮤니티 투표를 통한 탈중앙화 신원 검증을 가능하게 하는 오라클 시스템에 대해 알아보세요.

## 오라클이란?

오라클은 투표를 사용하여 DID(탈중앙화 신원) 속성을 검증하는 다자간 검증 시스템입니다. KYC와 같은 정부 속성에 제한된 기존 신원 시스템과 달리, 오라클은 커뮤니티 기반 인증을 통해 사용자 신원 속성의 확장을 가능하게 합니다.

## DID 속성 검증

오라클은 Ratel의 DID 발급자 기능의 기반 역할을 하며, 커뮤니티가 다양한 사용자 속성을 검증하고 증명할 수 있도록 합니다:

- **정부 속성**: 기존 KYC 검증
- **전문 자격증명**: 기술, 인증, 경력
- **평판 지표**: 커뮤니티 지위, 전문 영역
- **소셜 증명**: 검증된 소속, 회원
- **사용자 지정 속성**: 커뮤니티가 가치 있다고 판단하는 모든 속성
