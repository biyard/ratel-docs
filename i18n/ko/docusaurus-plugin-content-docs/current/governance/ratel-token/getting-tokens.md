# RATEL 토큰 획득

RATEL 토큰을 획득하는 다양한 방법입니다.

## 참여를 통해 획득

### 활발한 기여
- 고품질 게시물 작성
- 숙의 참여
- 스페이스 중재
- 설문조사 완료
- DID 속성 검증

### 보상 프로그램
- 일일 로그인 보너스
- 업적 배지
- 추천 보상
- 얼리 어답터 보너스

## 토큰 구매

2025년 2분기 출시 예정:
- 탈중앙화 거래소(Uniswap, Sushiswap)
- 중앙화 거래소(발표 예정)
- 신용카드로 직접 구매
- 얼리 서포터를 위한 토큰 사전판매

## 에어드랍

에어드랍 자격:
- 초기 플랫폼 사용자
- 활발한 커뮤니티 구성원
- 거버넌스 참여자
- 파트너 커뮤니티

자세한 내용은 [에어드랍](/rewards/airdrops/intro)을 확인하세요.

## 토큰 안전

⚠️ **중요**:
- 공식 소스만 사용
- 컨트랙트 주소 확인
- 사기 주의
- 개인 키 절대 공유 금지

공식 채널:
- 웹사이트: ratel.foundation
- 트위터: @RatelFoundation
- 디스코드: (링크 발표 예정)
