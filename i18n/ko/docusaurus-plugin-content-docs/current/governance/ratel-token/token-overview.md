# RATEL 토큰 개요

RATEL 토큰은 Ratel 생태계와 거버넌스를 구동합니다.

## 토큰 유틸리티

### 거버넌스 권한
- 오라클 검증에 투표
- 플랫폼 결정에 투표
- 변경사항 제안
- 투표권 위임

### 플랫폼 혜택
- 프리미엄 기능 액세스
- 수수료 할인
- 우선 지원
- 독점 콘텐츠

### 스테이킹 보상
출시 예정:
- 보상을 위한 토큰 스테이킹
- 플랫폼 수수료에서 수익
- 유동성 공급
- 검증자 보상

## 토큰 이코노믹스

### 총 공급량
토큰 출시 시 상세 내용 발표 예정

### 분배
- 커뮤니티 보상: 40%
- 개발팀: 20%
- 재무: 20%
- 생태계 성장: 15%
- 초기 판매: 5%

### 베스팅
- 팀 토큰: 4년 베스팅
- 어드바이저 토큰: 2년 베스팅
- 커뮤니티 보상 락업 없음

## 토큰 출시

예정: 2025년 2분기

출시 전:
- 화이트리스트 등록
- 커뮤니티 판매
- 유동성 공급
- 거래소 상장
