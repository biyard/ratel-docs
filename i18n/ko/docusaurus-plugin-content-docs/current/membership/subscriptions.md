# 구독 관리

멤버십 구독 및 관리 방법입니다.

## 구독하기

2025년 2분기 출시 예정:

1. 설정 → **멤버십**
2. 등급 선택
3. 결제 세부정보 입력
4. 구독 확인

## 결제 수단

- 신용/직불 카드
- 암호화폐(RATEL 토큰)
- PayPal
- Stripe

## 구독 취소

1. 설정 → **멤버십**
2. **구독 취소** 클릭
3. 취소 확인
4. 기간 종료까지 액세스 유지

## 등급 변경

1. 설정 → **멤버십**
2. **플랜 변경** 클릭
3. 새 등급 선택
4. 일할 계산된 청구 조정
