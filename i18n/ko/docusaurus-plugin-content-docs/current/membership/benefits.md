# 멤버십 혜택

Ratel 프리미엄 멤버십의 혜택입니다.

## 무료 등급

모든 사용자가 받는 혜택:
- 무제한 게시물 및 댓글
- 무제한 스페이스 참여
- 기본 숙의 참여
- 표준 분석
- 이메일 지원

## 프리미엄 등급

출시 예정: 월 $9.99

추가 혜택:
- 고급 분석
- 우선 지원
- 독점 배지
- 광고 없는 경험
- 조기 기능 액세스
- 업로드 한도 증가
- 사용자 지정 테마
- 프리미엄 스페이스 액세스

## 서포터 등급

출시 예정: 월 $29.99

모든 프리미엄 혜택에 추가:
- 거버넌스 투표 부스트
- 크리에이터 수익 공유
- 팀에 직접 액세스
- 베타 기능 테스트
- 이벤트 우선 액세스
- 사용자 지정 스페이스 기능
