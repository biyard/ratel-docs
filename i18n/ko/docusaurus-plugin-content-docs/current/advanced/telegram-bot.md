# Telegram 봇

알림 및 업데이트를 위해 Ratel을 Telegram과 통합하세요.

:::info 출시 예정
Telegram 봇 통합은 **2025년 3분기**로 예정되어 있습니다. 이 문서는 계획된 기능을 설명합니다.
:::

## Ratel Telegram 봇이란?

Ratel Telegram 봇은 플랫폼 기능을 Telegram으로 직접 가져와 다음을 수행할 수 있습니다:

- 실시간 알림 수신
- 게시물 보기 및 생성
- 숙의 참여
- 스페이스 업데이트 받기
- 계정 관리
- 커뮤니티와 상호작용

## 기능

### 알림

다음에 대한 즉시 알림 받기:
- 스페이스의 새 게시물
- 멘션 및 답글
- 다이렉트 메시지
- 숙의 업데이트
- 오라클 투표 요청
- 토큰 보상
- 중재 조치

### 콘텐츠 관리

**콘텐츠 보기**
- 최근 게시물 탐색
- 스레드 읽기
- 스페이스 활동 보기
- 숙의 결과 확인

**콘텐츠 생성**
- 텍스트 업데이트 게시
- 이미지 공유
- 투표 생성
- 토론에 답글

### 숙의 참여

**설문 응답**
- 설문 알림 수신
- 봇을 통해 답변 제출
- 투표 결과 보기
- 숙의 요약 받기

**토론**
- 숙의 토론 참여
- 의견 공유
- 게시물에 반응
- 합의 추적

### 스페이스 관리

**멤버 관리**
- 가입 요청 승인
- 권한 관리
- 멤버 목록 보기
- 공지사항 보내기

**중재**
- 신고된 콘텐츠 검토
- 중재 조치 취하기
- 신고 관리
- 사용자 차단/음소거

### 오라클 투표

**검증 알림**
- 전문 분야의 새 검증 요청
- 투표 알림
- 결과 알림
- 보상 업데이트

**빠른 투표**
- 증거 검토
- 투표하기
- 댓글 추가
- 투표 이력 추적

## Telegram 연결하기

### 1단계: 봇 찾기

Telegram에서 `@RatelBot`을 검색하거나 클릭:
```
https://t.me/RatelBot
```

### 2단계: 봇 시작

`/start`를 전송하여 시작:

```
👋 Ratel Bot에 오신 것을 환영합니다!

Ratel 계정을 연결하여 알림을 받고
Telegram에서 직접 스페이스를 관리하세요.

계정을 연결하려면 /connect를 입력하세요.
```

### 3단계: 계정 연결

1. `/connect` 전송
2. 인증 링크 클릭
3. Ratel에 로그인
4. Telegram 액세스 권한 부여
5. 확인 코드 수신
6. Telegram에 코드 입력

```
🎉 성공적으로 연결되었습니다!

이제 다음을 수행할 수 있습니다:
- /settings - 알림 구성
- /spaces - 스페이스 보기
- /help - 모든 명령 보기
```

## 봇 명령

### 계정 명령

**`/connect`**
- Ratel 계정 연결

**`/disconnect`**
- Telegram 연결 해제

**`/profile`**
- 프로필 통계 보기

**`/settings`**
- 알림 기본 설정 구성

### 콘텐츠 명령

**`/recent`**
- 스페이스의 최근 게시물 보기

**`/post [message]`**
- 새 게시물 생성

**`/spaces`**
- 스페이스 목록

**`/space [name]`**
- 스페이스 세부 정보 및 최근 활동 보기

**`/mentions`**
- 최근 멘션 보기

### 숙의 명령

**`/deliberations`**
- 스페이스의 활성 숙의

**`/vote [id]`**
- 숙의 참여

**`/results [id]`**
- 숙의 결과 보기

### 오라클 명령

**`/verify`**
- 대기 중인 검증 요청 확인

**`/cast [id] [approve/reject]`**
- 검증에 투표

**`/attestations`**
- 검증된 속성 보기

### 중재 명령

**`/reports`**
- 대기 중인 신고 보기

**`/moderate [action] [id]`**
- 중재 조치 취하기

**`/ban [username] [reason]`**
- 스페이스에서 사용자 차단

**`/mute [username] [duration]`**
- 사용자 일시적으로 음소거

### 유틸리티 명령

**`/help`**
- 모든 사용 가능한 명령 표시

**`/search [query]`**
- 게시물 및 스페이스 검색

**`/stats`**
- 활동 통계 보기

**`/notifications`**
- 알림 상태 확인

## 알림 설정

### 알림 사용자 지정

받을 알림 구성:

```
/settings

⚙️ 알림 설정

게시물 및 댓글:
🔔 스페이스의 새 게시물: 켜짐
🔔 게시물에 대한 답글: 켜짐
🔔 멘션: 켜짐
🔔 게시물 좋아요: 꺼짐

숙의:
🔔 새 숙의: 켜짐
🔔 투표 알림: 켜짐
🔔 결과: 켜짐

오라클:
🔔 검증 요청: 켜짐
🔔 투표 결과: 켜짐
🔔 보상: 켜짐

메시지:
🔔 다이렉트 메시지: 켜짐
🔔 그룹 초대: 켜짐

설정을 토글하려면 탭하세요
```

### 조용한 시간

알림이 일시 중지되는 시간 설정:

```
/quiethours

🌙 조용한 시간

시작: 23:00
종료: 08:00
시간대: America/New_York
긴급 알림: 켜짐

조용한 시간 동안 알림이 대기열에 추가되고
조용한 시간이 끝나면 전달됩니다.
```

### 우선순위 알림

중요한 알림은 조용한 시간을 무시합니다:
- 긴급 스페이스 공지사항
- 보안 알림
- 중요한 중재 문제
- 시간에 민감한 투표 마감일

## 봇 사용하기

### 게시물 생성

**텍스트 게시물:**
```
/post 방금 Crypto Policy 스페이스에 가입했습니다!
토론에 참여하게 되어 기쁩니다. 🚀
```

**이미지 게시물:**
1. 봇에 이미지 전송
2. `/post` 명령으로 캡션 추가
3. 대상 스페이스 선택

**투표:**
```
/poll

📊 투표 생성

질문: 다음에 무엇을 논의해야 할까요?
옵션 1: DeFi 규제
옵션 2: NFT 표준
옵션 3: DAO 거버넌스
옵션 4: 프라이버시 기술

기간: 7일
스페이스: Crypto Policy

확인하려면 전송
```

## 스페이스 통합

### 스페이스 전용 봇

스페이스를 위한 전용 봇 생성:

1. 설정 → **통합** → **Telegram**
2. "스페이스 봇 생성" 클릭
3. 권한 구성
4. Telegram 그룹에 봇 초대
5. 봇이 스페이스 업데이트 관리

### Telegram 그룹 동기화

Telegram 그룹을 스페이스와 동기화:

**기능:**
- 양방향 메시지 동기화
- 멤버 역할 매핑
- 중재 동기화
- 공지사항 미러링

**설정:**
1. Telegram 그룹 생성
2. @RatelBot을 관리자로 추가
3. `/sync` 명령
4. Ratel 스페이스 선택
5. 동기화 규칙 구성

### 자동 공지사항

Telegram에 공지사항 예약:

```
/announce

📢 공지사항 예약

메시지: 내일 오후 2시 UTC에 커뮤니티 통화!
스페이스: Crypto Policy
Telegram 그룹: 모두
예약: 2025-03-21 10:00 UTC

확인하여 예약
```

## 프라이버시 및 보안

### 데이터 프라이버시

봇은:
- DM에 종단 간 암호화 사용
- 최소 데이터 저장 (Telegram ID + Ratel 사용자 ID)
- 봇에 전송되지 않은 Telegram 메시지 액세스 안 함
- Telegram의 개인정보 보호정책 준수

### 보안 기능

**2단계 명령**
- 민감한 작업에는 확인 필요
- 차단/음소거 작업에는 이유 필요
- 검증 투표는 재고할 수 있음

**속도 제한**
- 스팸 방지
- 시간당 10개 게시물
- 시간당 50개 명령

**감사 로그**
- 모든 봇 작업 추적
- 명령 이력 검토
- 로그 내보내기

### 연결 해제

계정 연결을 해제하려면:

```
/disconnect

⚠️ Telegram 연결 해제

다음이 수행됩니다:
- 모든 알림 중지
- Telegram 통합 제거
- 봇 액세스 취소
- Ratel 계정 유지

계속하려면 CONFIRM을 입력하세요
```

## 문제 해결

**봇이 응답하지 않나요?**
- 봇이 차단되었는지 확인
- `/start`를 다시 시도
- `/connect`로 계정 재연결

**알림이 작동하지 않나요?**
- `/settings` 확인
- 조용한 시간 확인
- 스페이스 알림이 활성화되었는지 확인

**게시할 수 없나요?**
- 계정이 연결되었는지 확인
- 스페이스 권한 확인
- 속도 제한 검토

**명령이 작동하지 않나요?**
- 봇 업데이트 (`/start` 전송)
- 명령 구문 확인
- 지원 문의
