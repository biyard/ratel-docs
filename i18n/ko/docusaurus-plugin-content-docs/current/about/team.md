# 팀

Ratel을 만든 사람들을 만나보세요.

## 핵심 팀

Ratel은 참여형 거버넌스에 대한 공유 비전으로 뭉친 다양한 개발자, 정책 전문가, 블록체인 전문가 및 커뮤니티 빌더 팀이 구축합니다.

### 개발

**엔지니어링 팀**
- Rust, TypeScript 및 React 전문 풀스택 개발자
- Web3 전문 지식을 갖춘 블록체인 개발자
- 플랫폼 안정성을 보장하는 DevOps 엔지니어
- 사용자 데이터를 보호하는 보안 전문가

**제품 팀**
- 기능 개발을 안내하는 제품 관리자
- 직관적인 경험을 만드는 UX/UI 디자이너
- 품질 표준을 유지하는 QA 엔지니어

### 정책 및 거버넌스

**정책 전문가**
- 암호화폐 업계 경험이 있는 규제 전문가
- 규정 준수를 탐색하는 법률 고문
- 거버넌스 모델을 연구하는 정치학자
- 토큰 경제학을 분석하는 경제학자

**커뮤니티 관리자**
- 건강한 토론을 촉진하는 중재자
- 참여를 늘리는 커뮤니티 빌더
- 사용자를 교육하는 콘텐츠 크리에이터
- 멤버를 돕는 지원 전문가

### 운영

**비즈니스 팀**
- 성장을 계획하는 전략 리더
- 협업을 구축하는 파트너십 관리자
- 인지도를 높이는 마케팅 전문가
- 리소스를 관리하는 재무 전문가

## 자문 위원회

:::info 출시 예정
업계 리더, 정책 입안자 및 학계로 구성된 자문 위원회를 구성하고 있습니다. 세부 사항은 발표될 예정입니다.
:::

## 오픈 소스 기여자

Ratel은 커뮤니티와 함께, 커뮤니티를 위해 구축됩니다. 다음을 통해 플랫폼 개선을 돕는 많은 오픈 소스 기여자에게 감사드립니다:

- 코드 기여
- 문서 개선
- 버그 보고 및 수정
- 기능 제안
- 번역
- 디자인 에셋

### 기여하기

기여에 관심이 있으신가요? 방법은 다음과 같습니다:

**코드 기여**
1. [github.com/biyard/ratel](https://github.com/biyard/ratel) 방문
2. 미해결 이슈 확인
3. 저장소 포크
4. 풀 리퀘스트 제출

**문서**
1. [github.com/biyard/ratel-docs](https://github.com/biyard/ratel-docs) 방문
2. 기존 문서 개선
3. 새 가이드 추가
4. 오타 및 오류 수정

**커뮤니티 지원**
- Discord에서 질문에 답변
- 신규 사용자가 시작하도록 돕기
- 전문 지식 공유
- 토론 중재

**콘텐츠 생성**
- 블로그 게시물 작성
- 튜토리얼 생성
- 비디오 제작
- 그래픽 디자인

**영향**

## 문화

### 원격 우선

우리는 최고의 인재가 전 세계에 분산되어 있다고 믿습니다. 우리 팀은 시간대를 넘어 협업하며 필요할 때 동기적으로 만나고 비동기적으로 작업합니다.

**커뮤니케이션 도구:**
- 팀 채팅을 위한 Discord
- 코드 협업을 위한 GitHub
- 문서를 위한 Notion
- 화상 통화를 위한 Zoom

### 투명성

투명성을 기반으로 구축된 플랫폼으로서 우리는 설교하는 것을 실천합니다:
- 공개 로드맵
- 공개 개발
- 정기적인 커뮤니티 업데이트
- 투명한 의사결정

### 지속적인 학습

블록체인 공간은 빠르게 진화합니다. 우리는 다음을 통해 지속적인 학습에 투자합니다:
- 주간 지식 공유 세션
- 컨퍼런스 참석
- 온라인 과정 및 인증
- 내부 워크숍

### 워크-라이프 밸런스

지속 가능한 플랫폼을 구축하려면 지속 가능한 관행이 필요합니다:
- 크런치 문화 없음
- 합리적인 근무 시간
- 휴가 권장
- 정신 건강 지원
