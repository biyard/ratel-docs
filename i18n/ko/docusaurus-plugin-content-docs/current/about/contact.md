# 문의하기

Ratel 팀에 연락하세요.

## 연락 방법

도와드리겠습니다! 귀하의 필요에 가장 적합한 방법을 선택하세요.

### 일반 문의

**이메일:** hi@biyard.co
**응답 시간:** 영업일 기준 2-3일 이내

Ratel, 기능 또는 시작에 대한 일반적인 질문은 여기로 문의하세요.

### 지원

**이메일:** support@biyard.co
**Discord:** [#support 채널 가입](https://discord.gg/ratel)
**응답 시간:** 24-48시간 이내

기술적 문제, 계정 문제 또는 플랫폼 질문은 여기로 문의하세요.

참고: [지원 가이드](/faq/support)

### 비즈니스 개발

**이메일:** partnerships@biyard.co
**응답 시간:** 48-72시간 이내

파트너십 기회, 통합 또는 비즈니스 협력은 여기로 문의하세요.

참고: [파트너 및 협력자](./partners)

### 미디어 및 언론

**이메일:** press@biyard.co
**응답 시간:** 24-48시간 이내

언론 문의, 인터뷰 요청 또는 미디어 리소스는 여기로 문의하세요.

참고: [프레스 키트](./press-kit)

### 보안

**이메일:** security@biyard.co
**응답 시간:** 중요 문제의 경우 24시간 이내

보안 취약점 또는 우려 사항 보고는 여기로 문의하세요.

:::caution 중요
보안 취약점의 경우 공개 GitHub 이슈를 생성하지 마세요. security@biyard.co로 세부 사항을 이메일로 보내주세요.

참고: [보안 보고](/security/reporting)

## 문제 보고

### 버그 보고

**플랫폼 버그:**
1. [GitHub 이슈](https://github.com/biyard/ratel/issues) 확인
2. 상세한 버그 보고 생성
3. 재현 단계 포함
4. 해당하는 경우 스크린샷 추가

**보안 취약점:**
1. security@biyard.co로 이메일
2. 상세한 설명 포함
3. 공개적으로 공개하지 마세요
4. 패치 시간 허용

### 콘텐츠 문제

**부적절한 콘텐츠:**
1. 앱 내 보고 기능 사용
2. 또는 이메일: moderation@biyard.co
3. 게시물/사용자 URL 포함
4. 위반 설명

참고: [커뮤니티 가이드라인](/legal/guidelines) • [보고](/security/reporting)

### 사기 및 사기

**의심스러운 사기:**
1. 이메일: security@biyard.co
2. 증거 포함
3. 사용자/콘텐츠 세부 사항
4. 사기꾼과 관여하지 마세요

:::warning 절대 공유하지 마세요
Ratel 팀은 절대 요청하지 않습니다:
- 개인 키 또는 시드 문구
- 비밀번호
- 지원에 대한 결제
- 개인 금융 정보
:::
