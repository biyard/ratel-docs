# 데이터 보호

Ratel은 강력한 기술적 및 조직적 조치를 통해 귀하의 개인 데이터를 보호하기 위해 최선을 다하고 있습니다.

## 데이터 분류

당사는 데이터를 적절한 보호 수준으로 범주화합니다:

- **공개 데이터**: 공개적으로 표시되는 프로필 정보
- **개인 데이터**: 이메일, 인증 자격 증명
- **민감한 데이터**: 금융 정보, 개인 키
- **사용 데이터**: 분석 및 행동 데이터

## 암호화

### 저장 시
- 모든 저장된 데이터에 대한 AES-256-GCM 암호화
- 키 관리를 위한 AWS Key Management Service(KMS)
- 개인 키를 위한 클라이언트 측 암호화

### 전송 중
- 모든 네트워크 통신에 대한 TLS 1.3
- 모바일 앱을 위한 인증서 고정
- 안전한 WebSocket 연결

## 액세스 제어

- **인증**: 다중 인증(MFA) 지원
- **권한 부여**: 역할 기반 액세스 제어(RBAC)
- **세션 관리**: 자동 만료 기능이 있는 안전한 토큰
- **API 키**: 순환 정책이 있는 암호화된 저장

## 데이터 보관

- **활성 계정**: 계정이 활성 상태인 동안 데이터 보관
- **삭제된 계정**: 30일 유예 기간 후 영구 삭제
- **백업**: 암호화와 함께 90일 동안 보관
- **법적 보류**: 법률에서 요구하는 경우 보관 기간 연장

## 사용자 개인정보 제어

귀하는 다음을 수행할 수 있습니다:
- 이동 가능한 형식으로 데이터 내보내기
- 계정 및 모든 관련 데이터 삭제
- 프로필 정보의 가시성 제어
- 제3자 데이터 공유 설정 관리

## 데이터 침해 대응

데이터 침해가 발생하는 경우:

1. 즉시 격리 및 조사
2. 72시간 이내에 사용자 통지
3. 요구되는 경우 규제 기관 통지
4. 투명한 공개 공지
5. 개선 및 예방 조치

## 규정 준수

- **GDPR**: EU 일반 데이터 보호 규정
- **CCPA**: 캘리포니아 소비자 개인정보 보호법
- **PIPEDA**: 캐나다 개인정보 보호법 준수
