# 스페이스 참여

관심 있는 스페이스를 발견하고 참여하는 방법을 알아보세요.

## 스페이스 찾기

### 스페이스 탐색
탐색 페이지를 통해 사용 가능한 스페이스 찾아보기.

### 스페이스 검색
검색 기능을 사용하여 주제나 키워드로 스페이스 찾기.

### 초대 링크
스페이스 크리에이터의 초대 링크를 사용하여 비공개 스페이스 참여.

## 참여 프로세스

### 공개 스페이스
1. 스페이스 페이지로 이동
2. "스페이스 참여" 클릭
3. 즉시 참여 시작

### 비공개 스페이스
1. 스페이스 크리에이터가 제공한 초대 링크 클릭
2. 필요한 경우 로그인 또는 가입
3. 필요한 인증 완료
4. 스페이스 초대 수락

## 신원 인증

일부 스페이스는 신원 인증이 필요할 수 있습니다:
1. 메시지가 표시되면 인증 코드 입력
2. 인증 프로세스 완료
3. 스페이스 참여 진행

## 사전 참여 설문조사

특정 스페이스는 사전 참여 설문조사를 요청할 수 있습니다:
1. 설문조사 질문 검토
2. 응답 제공
3. 설문조사 제출
4. 필요한 경우 승인 대기

## 다음 단계

스페이스 참여 후:
- 스페이스 개요 및 가이드라인 검토
- 커뮤니티에 자신 소개
- 토론 참여 시작
- 거버넌스 활동 참여
