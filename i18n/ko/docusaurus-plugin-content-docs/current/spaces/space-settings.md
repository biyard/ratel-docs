# 스페이스 설정 및 권한

스페이스 설정을 구성하고 권한을 효과적으로 관리하세요.

## 일반 설정

### 스페이스 정보
- 스페이스 이름 업데이트
- 설명 편집
- 카테고리/태그 변경

### 가시성 설정
- 공개: 누구나 보고 참여 가능
- 비공개: 초대 전용
- 목록에 없음: 링크를 통해서만 액세스 가능

## 구성원 권한

### 역할 유형

#### 소유자
- 스페이스에 대한 전체 제어
- 스페이스 삭제 가능
- 모든 설정 관리

#### 중재자
- 토론 중재
- 구성원 관리
- 스페이스 삭제 불가

#### 구성원
- 토론 참여
- 콘텐츠 보기
- 제안에 투표

### 권한 설정
- 게시물 작성 권한
- 댓글 권한
- 설문조사 참여
- 파일 업로드 권한

## 개인정보 설정

### 익명 참여
익명 게시 및 댓글 활성화 또는 비활성화.

### 데이터 가시성
다음 대상에게 표시되는 데이터 제어:
- 스페이스 구성원
- 공개 뷰어
- 중재자

## 알림 설정

다음에 대한 알림 구성:
- 새 구성원
- 새 게시물
- 설문조사 응답
- 중재 필요

## 고급 설정

### 통합 설정
- API 액세스
- 웹훅 구성
- 내보내기 설정

### 삭제 및 보관
- 비활성 스페이스 보관
- 스페이스 영구 삭제
- 스페이스 데이터 내보내기
