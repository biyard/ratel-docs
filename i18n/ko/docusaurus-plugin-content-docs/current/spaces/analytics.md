# 분석

분석을 사용하여 커뮤니티를 이해하고 성장시키세요.

## 사용 가능한 분석

### 성장 지표
- 총 구성원 수
- 신규 구성원(일일/주간/월간)
- 구성원 유지율
- 성장 추세

### 참여 지표
- 일일 게시물 수
- 게시물당 댓글 수
- 활동 구성원
- 참여율

### 콘텐츠 성과
- 인기 게시물
- 가장 참여도가 높은 주제
- 최적의 게시 시간
- 콘텐츠 유형 성과

## 분석 액세스

1. 스페이스로 이동
2. **분석** 탭 클릭
3. 대시보드 보기
4. 데이터 내보내기(CSV)

## 주요 인사이트

### 추적할 것
- 구성원 활동 패턴
- 콘텐츠 선호도
- 피크 참여 시간
- 트렌딩 주제

### 무시할 것
- 허영 지표
- 단기 변동
- 이상치 이벤트
- 실행 불가능한 데이터

## 분석 활용

### 콘텐츠 전략
- 구성원이 활동할 때 게시
- 효과가 있는 것을 더 많이 만들기
- 새로운 형식 실험
- 게시물 길이 최적화

### 커뮤니티 성장
- 성장 소스 식별
- 이탈 지점 파악
- 유지율 개선
- 이상적인 구성원 타겟팅

### 숙의 인사이트
- 설문조사 참여율
- 의견 분포
- 합의 형성
- 토론 품질

## 고급 분석

출시 예정:
- 감성 분석
- 주제 모델링
- 구성원 여정 매핑
- 예측 분석
