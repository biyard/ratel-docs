# 스페이스 만들기

관심사나 거버넌스 목표를 중심으로 커뮤니티를 구축하기 위해 자신만의 스페이스를 만드세요.

## 전제 조건

스페이스를 만들기 전에 다음을 확인하세요:
- Ratel 계정
- 스페이스의 명확한 목적
- 스페이스 관리에 대한 이해

## 스페이스 만들기

### 1단계: 스페이스 만들기로 이동
1. 홈 화면에서 "게시물 작성" 버튼 클릭
2. 게시물의 제목 및 설명 입력
3. 스페이스 유형 선택(숙의, 커뮤니티 등)

### 2단계: 스페이스 설정 구성

#### 개요 추가
- 스페이스의 목적에 대한 컨텍스트 제공
- 목표 정의
- 참여 가이드라인 설정

#### 파일 업로드
- 지원 문서 첨부
- 참고 자료 추가
- 관련 리소스 포함

#### 사전 설문조사 구성(선택 사항)
- 참가자를 위한 질문 작성
- 배경 정보 수집
- 기준 의견 수집

#### 보드 설정
- 전용 토론 보드 생성
- 보드 카테고리 구성
- 게시 규칙 설정

#### 구성원 초대
- 스페이스에 초기 구성원 추가
- 구성원 역할 및 권한 설정
- 초대 설정 구성

#### 패널 구성(선택 사항)
- 구조화된 숙의를 위한 패널 구성원 설정
- 패널 역할 정의
- 패널 권한 구성

### 3단계: 개인정보 설정

#### 익명 참여(선택 사항)
민감한 토론을 위해 참가자가 표시 이름을 숨길 수 있도록 허용.

#### 가시성 설정(선택 사항)
생성 후 스페이스 가시성을 변경할 수 있는지 제어.

#### 비공개 스페이스
초대된 참가자만 액세스를 제한.

## 스페이스 게시

1. 모든 설정 검토
2. "스페이스 게시" 클릭
3. 초대된 구성원에게 초대 이메일 자동 전송

**중요:** 스페이스가 시작되면 새 참가자는 참여할 수 없습니다.
