# 스페이스 관리

스페이스를 효과적으로 관리하고 중재하는 방법을 알아보세요.

## 스페이스 대시보드

스페이스 대시보드에 액세스하여:
- 스페이스 분석 보기
- 구성원 관리
- 토론 모니터링
- 설문조사 검토

## 스페이스 시작

### 시작하기 전에
- 초대된 모든 구성원이 참여했는지 확인
- 스페이스 설정이 올바른지 확인
- 가이드라인 및 규칙 검토

### 스페이스 시작
1. 스페이스 대시보드로 이동
2. "스페이스 시작" 클릭
3. 작업 확인

**중요:** 일단 시작되면 새 참가자는 참여할 수 없습니다.

## 구성원 관리

### 구성원 추가
- 초대 링크 보내기
- 참여 요청 승인
- 이메일을 통해 대량 초대

### 역할 관리
- 중재자 역할 할당
- 권한 설정
- 액세스 수준 구성

### 구성원 제거
- 비활성 구성원 제거
- 필요한 경우 사용자 차단
- 위반 처리

## 토론 중재

### 게시물 중재
- 신고된 콘텐츠 검토
- 보류 중인 게시물 승인(활성화된 경우)
- 부적절한 콘텐츠 제거

### 커뮤니티 가이드라인
- 스페이스 규칙 시행
- 위반 사항 해결
- 건전한 토론 유지

## 설문조사 및 데이터

### 설문조사 만들기
1. 설문조사 섹션으로 이동
2. 사전 설문조사 또는 최종 설문조사 만들기
3. 참가자에게 게시

### 결과 검토
1. 데이터 내보내기로 이동
2. 원시 데이터 내보내기(CSV)
3. 응답 분석

## 스페이스 설정

설정에 액세스하여:
- 스페이스 정보 업데이트
- 개인정보 설정 수정
- 권한 변경
- 알림 구성
