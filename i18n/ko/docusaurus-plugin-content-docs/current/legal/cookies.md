# 쿠키 정책

Ratel의 쿠키 및 유사 기술 사용 방법에 대해 알아보세요.

## 쿠키란?

쿠키는 웹사이트 방문 시 기기에 저장되는 작은 텍스트 파일입니다. 쿠키를 통해:
- 로그인 상태 유지
- 설정 기억
- 사용 패턴 분석
- 사용자 경험 개선

## 사용하는 쿠키

### 필수 쿠키

**인증 쿠키**
- 로그인 상태 유지
- 세션 관리
- 보안 토큰

**기능 쿠키**
- 언어 설정
- 테마 선택
- 레이아웃 설정

### 분석 쿠키

**사용 메트릭**
- 페이지 조회수
- 클릭 패턴
- 기능 사용
- 성능 데이터

### 제3자 쿠키

**인증 제공자**
- Google OAuth
- Web3 지갑
- Firebase

**분석 서비스**
- 사용 통계
- 오류 추적
- 성능 모니터링

## 쿠키 관리

### 브라우저 설정

대부분의 브라우저에서 쿠키를 제어할 수 있습니다:
- 모든 쿠키 차단
- 제3자 쿠키만 차단
- 쿠키 삭제
- 쿠키 통지 설정

### 플랫폼 설정

설정 → **개인정보** → **쿠키 설정**에서:
- 분석 쿠키 선택 해제
- 광고 쿠키 차단
- 설정 내보내기

### 영향

쿠키를 차단하면:
- 일부 기능이 작동하지 않을 수 있음
- 설정이 저장되지 않을 수 있음
- 로그인 상태가 유지되지 않을 수 있음

## 기타 기술

### 로컬 스토리지

- 지갑 연결
- 임시 데이터
- 캐시

### 세션 스토리지

- 탭별 데이터
- 임시 상태
- 양식 데이터
