# 보상 획득

Ratel 커뮤니티 참여로 보상을 획득하세요.

## 보상 유형

- **RATEL 토큰**: 플랫폼 거버넌스 토큰
- **포인트**: 기능으로 교환 가능
- **배지**: 업적 표시
- **NFT**: 독점 수집품

## 획득 방법

### 일일 활동
- 일일 로그인: 10 포인트
- 고품질 게시물 작성: 50 포인트
- 토론 참여: 25 포인트
- 설문조사 완료: 100 포인트

### 업적
- 첫 게시물: 100 포인트
- 팔로워 100명: 500 포인트
- 스페이스 생성: 1000 포인트
- DID 검증: 2000 포인트

### 숙의
- 참여: 200 포인트
- 사전 및 사후 설문조사 완료: +100 포인트
- 고품질 기여: +500 포인트

### 오라클 검증
- 검증에 투표: 50 토큰
- 정확한 투표: +50 토큰 보너스
- 신뢰할 수 있는 검증자 되기: 지속적인 보상

## 보상 교환

포인트 사용처:
- 프리미엄 기능
- 사용자 지정 배지
- 이벤트 티켓
- 프로필 사용자 지정
