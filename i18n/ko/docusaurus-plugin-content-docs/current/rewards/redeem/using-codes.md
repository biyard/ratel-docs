# 보상 코드 사용하기

특별 코드를 교환하여 보상을 받으세요.

## 코드를 찾을 수 있는 곳

- 이메일 캠페인
- 소셜 미디어 콘테스트
- 파트너 프로모션
- 커뮤니티 이벤트
- 컨퍼런스 경품

## 코드 교환하기

1. 설정 → **보상**
2. "코드 교환" 클릭
3. 코드 입력
4. "교환" 클릭
5. 보상 수령

## 코드 유형

- **일회용**: 1회 사용
- **다중 사용**: 제한된 수량
- **시간 제한**: 날짜 후 만료
- **사용자 제한**: 계정당 1회

## 문제 해결

**코드가 작동하지 않나요?**
- 오타 확인
- 만료되지 않았는지 확인
- 이미 사용하지 않았는지 확인
- 자격 요구사항 확인
