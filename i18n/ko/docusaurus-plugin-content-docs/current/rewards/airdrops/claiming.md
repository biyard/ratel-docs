# 에어드랍 청구하기

RATEL 토큰을 청구하는 단계별 가이드입니다.

## 청구 전 준비

1. **지갑 설정**
   - MetaMask 또는 호환 지갑 설치
   - 시드 문구 보안
   - RATEL 토큰 컨트랙트 추가(주소 발표 예정)

2. **자격 확인**
   - ratel.foundation/airdrop 방문
   - 지갑 연결
   - 할당량 확인

## 청구 프로세스

1. ratel.foundation/airdrop로 이동
2. "지갑 연결" 클릭
3. 연결 승인
4. 할당량 검토
5. "토큰 청구" 클릭
6. 트랜잭션 승인
7. 가스 수수료 지불
8. 토큰 수령

## 중요 사항

- 청구 기간: 90일
- 청구되지 않은 토큰은 재무로 반환
- 가스 수수료는 청구자 부담
- 주소당 1회 청구

## 청구 후

- 지갑에서 토큰 확인
- 유동성 풀에 추가(선택 사항)
- 거버넌스 참여
- 보상을 위한 스테이킹
