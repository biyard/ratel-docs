# RATEL 토큰 에어드랍

얼리 서포터와 활발한 커뮤니티 구성원에게 보상합니다.

## 에어드랍 프로그램

RATEL 토큰 분배 대상:
- 얼리 어답터(2025년 2분기 이전)
- 활발한 커뮤니티 구성원
- 거버넌스 참여자
- 스페이스 크리에이터

## 자격

기준은 [자격](./eligibility)을 확인하세요.

## 청구

에어드랍 출시 시:
1. 지갑 연결
2. 자격 확인
3. 토큰 청구
4. 가스 수수료 지불

[청구 가이드](./claiming)를 참조하세요.
