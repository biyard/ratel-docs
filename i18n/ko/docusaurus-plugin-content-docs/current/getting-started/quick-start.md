# 빠른 시작 가이드

Ratel에 오신 것을 환영합니다! 이 가이드는 몇 분 안에 플랫폼을 시작하는 데 도움을 드립니다.

## 학습 내용

이 빠른 시작 가이드에서:
- Ratel 계정 만들기
- 메인 인터페이스 이해하기
- 첫 숙의 스페이스에 참여하거나 Fine-Art (d.AGIT) 탐색하기
- 커뮤니티와 교류 시작하기

## 사전 요구사항

시작하기 전에 다음을 확인하세요:
- 최신 웹 브라우저(Chrome, Firefox, Safari 또는 Edge)
- 유효한 이메일 주소 또는 Google 계정
- 인터넷 연결

## 1단계: 계정 만들기

1. [Ratel](https://dev.ratel.foundation) 방문
2. 오른쪽 상단의 **로그인** 클릭
3. 가입 방법 선택:
   - **이메일**: 이메일을 입력하고 인증 단계를 따르세요
   - **Google**: Google 계정으로 빠른 가입

자세한 지침은 [계정 만들기](./sign-up)를 참조하세요.

## 2단계: 프로필 완성하기

가입 후:
1. **표시 이름** 설정 - 다른 사람들이 보는 이름
2. 고유한 **사용자 이름** 선택 - 플랫폼에서의 핸들(예: @yourname)
3. **프로필 사진** 추가(선택사항이지만 권장)
4. 자기소개를 위한 간단한 **바이오** 작성

## 3단계: 플랫폼 탐색하기

### 메인 내비게이션

Ratel 인터페이스는 주요 영역으로 구성됩니다:

**상단 내비게이션 바**:
- **홈** - 개인화된 피드
- **멤버십** - 계정 및 구독 관리
- **테스트 리포트** - 플랫폼 테스트 정보 보기
- **스토리북** - UI 컴포넌트 탐색
- **언어 선택기** - 영어와 한국어 간 전환
- **테마 토글** - 라이트 모드와 다크 모드 선택
- **로그인/프로필** - 계정 액세스

**사이드바**:
- **게시물 작성** - 생각과 아이디어 공유
- **인기 프로모션** - 주요 스페이스 및 토론

### 피드

홈 피드 표시 내용:
- 참여한 스페이스의 최근 게시물
- 참여 가능한 숙의 주제
- 커뮤니티 토론 및 업데이트
- 참여자가 필요한 프로모션된 스페이스

## 4단계: 경로 선택하기

Ratel은 두 가지 주요 경험을 제공합니다:

### 숙의 및 정책 토론용

구조화된 정책 토론에 관심이 있다면:

1. **스페이스 탐색** - 피드에서 숙의 스페이스 찾기
2. **스페이스 참여** - 스페이스 카드를 클릭하고 "지금 참여" 선택
3. **사전 설문 완료** - 숙의 시작 전 초기 질문에 답변
4. **토론 참여** - 토론 게시판에서 의견 공유
5. **최종 설문 완료** - 숙의 후 피드백 제공

자신만의 숙의를 시작하려면 [스페이스 만들기](../spaces/creating-spaces)를 참조하세요.

### 미술품 거래 및 수집용 (Fine-Art / d.AGIT)

**d.AGIT** (Digital Assets Guardianship & Interchange Terminal)을 통한 미술품 거래에 관심이 있다면:

1. **미술품 탐색** - 이용 가능한 미술품 탐색(출시 예정)
2. **출처 보기** - 진위성을 위한 전자 미술품 기록(EAR) 확인
3. **미술품 검증** - 오라클 검증 상태 검토
4. **구매하기** - 스마트 계약을 통한 안전한 거래 완료

## 5단계: 커뮤니티 참여하기

### 게시물 작성하기

1. 사이드바에서 **게시물 작성** 클릭
2. 리치 텍스트 에디터로 콘텐츠 작성
3. 필요에 따라 미디어, 링크 또는 서식 추가
4. 대상 선택(공개 또는 스페이스별)
5. **게시** 클릭

### 토론 참여하기

- 공감하는 게시물에 **좋아요**
- 관점을 공유하기 위해 **댓글** 달기
- 흥미로운 토론 **공유**
- 좋아하는 콘텐츠를 만드는 사용자 **팔로우**

## 일반 작업

### 초대 받기

숙의 스페이스에 초대받았을 때:
1. 보류 중인 초대를 위해 **내 스페이스** 확인
2. 스페이스 목표 및 요구사항 검토
3. 초대 수락하여 참여
4. 필요한 자격 증명 검증 완료

### 알림 관리

플랫폼 활동으로 최신 정보 유지:
- 상단 바에서 알림 아이콘 클릭
- 초대, 멘션 및 업데이트 검토
- 설정에서 알림 기본 설정 구성

## 성공을 위한 팁

### 숙의 참여자용

- **주의 깊게 읽기** - 설문 전에 모든 배경 자료 검토
- **신중하게 생각하기** - 다양한 관점을 고려할 시간 갖기
- **예의 유지** - 의견 불일치에서도 존중하는 대화 유지
- **설문 완료** - 사전/사후 응답이 합의 측정에 도움

### 콘텐츠 제작자용

- **명확하게** - 간결하고 잘 구조화된 게시물 작성
- **출처 인용** - 증거로 주장 뒷받침
- **진정성 있게 참여** - 댓글과 질문에 응답
- **서식 사용** - 헤더와 목록으로 게시물을 읽기 쉽게

### 미술 시장 참여자용

- **철저히 검증** - 구매 전 모든 EAR 문서 확인
- **오라클 상태 검토** - 여러 당사자가 진위성을 검증했는지 확인
- **가격 이해** - AI 추천을 절대적 진실이 아닌 가이드로 사용
- **기록 보관** - 거래에 대한 자체 문서 유지
