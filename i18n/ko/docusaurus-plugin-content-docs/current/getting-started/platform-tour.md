# 플랫폼 둘러보기

Ratel 플랫폼의 포괄적인 투어를 통해 모든 기능을 발견하세요.

![Ratel 플랫폼](/img/ratel-homepage.png)

## 플랫폼 개요

Ratel은 직관적이고 강력하게 설계되었습니다. 이 투어는 플랫폼의 각 영역을 안내하여 최상의 경험을 할 수 있도록 도와드립니다.

## 상단 내비게이션 바

모든 페이지 상단의 내비게이션 바는 주요 기능에 빠르게 액세스할 수 있습니다:

### 홈
다음을 보여주는 개인화된 피드:
- 참여한 스페이스의 게시물
- 숙의 초대
- 커뮤니티 업데이트
- 인기 토론

### 멤버십
Ratel 구독 및 혜택 관리:
- 현재 멤버십 등급 보기
- 플랜 업그레이드 또는 다운그레이드
- 독점 기능 액세스
- 리워드 포인트 추적

### 테스트 리포트
플랫폼 테스트 및 품질 정보 액세스:
- 현재 테스트 커버리지 보기
- 플랫폼 상태 확인
- 최근 배포 검토
- 기능 가용성 확인

### 스토리북
Ratel UI 컴포넌트 라이브러리 탐색:
- 사용 가능한 컴포넌트 탐색
- 컴포넌트 변형 보기
- 인터랙티브 요소 테스트
- 디자인 시스템 이해

### 언어 선택기
지원되는 언어 간 전환:
- **EN** - 영어(English)
- **KO** - 한국어

언어 기본 설정이 저장되고 모든 페이지에 적용됩니다.

### 테마 토글
선호하는 시각적 테마 선택:
- **라이트 모드** - 밝고 높은 대비의 인터페이스
- **다크 모드** - 장시간 사용 시 눈의 피로 감소

테마 기본 설정은 계정에 저장됩니다.

### 로그인 / 프로필
계정 기능 액세스:
- **로그인하지 않음**: 클릭하여 로그인하거나 계정 만들기
- **로그인함**: 프로필, 설정 및 로그아웃 액세스

## 메인 콘텐츠 영역

중앙 영역은 현재 페이지에 따라 컨텍스트별 콘텐츠를 표시합니다.

### 홈 피드

홈 피드는 활동의 시간순 스트림을 보여줍니다:

**게시물 카드** 포함 사항:
- **작성자** - 프로필 사진 및 표시 이름
- **타임스탬프** - 게시물이 생성된 시간
- **콘텐츠** - 게시물 텍스트, 이미지 또는 미디어
- **참여 지표**:
  - 좋아요
  - 댓글
  - 공유
  - 북마크

**스페이스 카드** 표시 사항:
- **스페이스 뱃지** - 숙의 스페이스임을 나타냄
- **제목** - 스페이스 이름
- **설명** - 간략한 개요
- **생성자** - 스페이스를 만든 사람
- **상태** - 개방, 진행 중 또는 완료
- **참여 버튼** - 초대 수락 또는 참여 요청

### 게시물 인터랙션

각 게시물은 다양한 참여 방법을 제공합니다:

**좋아요** (👍)
- 콘텐츠에 대한 감사 표시
- 게시물 인기도에 기여
- 누가 좋아요를 눌렀는지 공개 표시 없음

**댓글** (💬)
- 생각 공유
- 토론 시작
- 다른 댓글에 답글 달기
- 스레드형 대화

**공유** (🔄)
- 피드에 재게시
- 특정 스페이스에 공유
- 코멘트 추가

**북마크** (🔖)
- 나중에 읽기 위해 저장
- 저장된 게시물 정리
- 본인에게만 비공개

## 사이드바

오른쪽 사이드바는 빠른 작업 및 하이라이트를 제공합니다:

### 게시물 작성

**게시물 작성**을 클릭하여 콘텐츠 공유:
1. 게시물 에디터 열기
2. 리치 텍스트로 메시지 작성
3. 서식, 링크 또는 미디어 추가
4. 가시성 선택(공개 또는 스페이스)
5. 피드에 게시

### 인기 프로모션

주요 스페이스 및 캠페인:
- **활성 숙의** - 진행 중인 토론에 참여
- **프로모션된 스페이스** - 참여자가 필요한 중요 주제
- **커뮤니티 이벤트** - 다가오는 활동

프로모션된 항목을 클릭하여 자세히 알아보거나 참여하세요.

## 스페이스

스페이스는 숙의와 토론을 위한 구조화된 환경입니다.

### 스페이스 유형

**숙의 스페이스**:
- 사전 설문 → 토론 → 사후 설문 워크플로
- 익명 또는 신원 공개 참여
- 중재된 토론
- 분석 및 보고

**팀 스페이스**:
- 조직별 영역
- 멤버 전용 액세스
- 협업 기능
- 공유 리소스

### 스페이스 수명 주기

1. **생성됨** - 스페이스 구성 중
2. **게시됨** - 참여자 참여 개방
3. **사전 설문** - 초기 의견 수집
4. **활성 숙의** - 토론 단계
5. **최종 설문** - 숙의 후 피드백
6. **완료됨** - 결과 검토 가능

### 스페이스 참여하기

흥미로운 스페이스를 발견하면:

1. **세부 정보 검토**:
   - 스페이스 목표 읽기
   - 참여 요구사항 확인
   - 타임라인 및 일정 보기
   - 필요한 약속 이해

2. **초대 수락**:
   - "지금 참여" 클릭
   - 자격 증명 검증 완료
   - 스페이스 규칙에 동의
   - 참여 확인

3. **사전 설문 완료**:
   - 초기 질문에 답변
   - 기준선 의견 공유
   - 인구통계 정보 제공(필요한 경우)
   - 응답 제출

4. **스페이스 입장**:
   - 토론 게시판 액세스
   - 배경 자료 읽기
   - 다른 참여자 보기(익명이 아닌 경우)
   - 참여 시작

## 토론 게시판

스페이스 내부의 조직화된 토론 영역:

### 게시판 구조

**주제** - 각 게시판은 특정 측면에 집중:
- 질문 또는 이슈 진술
- 배경 정보
- 참고 자료
- 토론 가이드라인

**게시물** - 토론에 대한 기여:
- 주장을 제시하는 원본 게시물
- 기존 게시물에 대한 답글
- 스레드형 대화
- 멀티미디어 지원

**중재** - 품질 유지:
- 생성자 감독
- 커뮤니티 신고
- 가이드라인 시행
- 예의 있는 담론

### 토론에 게시하기

게시판에 기여하려면:

1. **주제 선택** - 관련 토론 게시판 선택
2. **기존 게시물 읽기** - 현재 대화 이해
3. **게시물 작성**:
   - 명확한 주장 또는 질문
   - 뒷받침하는 증거 또는 추론
   - 존중하는 어조
   - 적절한 인용
4. **검토 및 게시** - 제출 전 명확성 확인

### 익명 참여

일부 스페이스는 익명 게시를 허용합니다:

- 신원이 다른 참여자에게 숨겨짐
- 생성자와 중재자는 익명 ID를 볼 수 있음
- 모든 기여는 익명으로 추적됨
- 설문 응답은 기밀 유지

민감한 주제에 대한 솔직한 토론을 장려합니다.

## 설문 및 투표

스페이스 내 구조화된 데이터 수집:

### 사전 설문

숙의 시작 전:
- **객관식** - 미리 정의된 옵션에서 선택
- **순위** - 선호도 순서 지정
- **리커트 척도** - 진술에 대한 동의 평가
- **개방형 응답** - 자유 텍스트 답변

목적: 토론이 견해에 영향을 미치기 전 기준선 의견 수집.

### 최종 설문

숙의 종료 후:
- **동일한 질문** - 사전/사후 비교 가능
- **의견 추적** - 견해 변화 측정
- **합의 분석** - 동의 영역 식별
- **보고** - 데이터 시각화 및 내보내기

목적: 숙의가 관점에 미친 영향 이해.

자세한 내용은 [프로필 설정](../guide/account/profile-settings)을 참조하세요.

## Fine-Art (d.AGIT) 플랫폼

**d.AGIT** (Digital Assets Guardianship & Interchange Terminal) - P2P 미술품 거래 기능(출시 예정):

### 미술품 탐색

- 갤러리 스타일 탐색
- 카테고리, 작가, 가격별 필터링
- 고해상도 이미지 보기
- 출처 및 검증 상태 확인

### 전자 미술품 기록(EAR)

완전한 미술품 문서:
- 창작 세부 정보
- 작가 정보
- 소유권 이력
- 상태 보고서
- 전문가 감정
- 가격 이력

### 검증 상태

오라클 풀 검증 표시:
- ✅ **검증됨** - 여러 당사자가 진위성 확인
- ⏳ **진행 중** - 검증 진행 중
- ❌ **미검증** - 아직 인증되지 않음
- ⚠️ **분쟁 중** - 검증 충돌

### 거래 프로세스

안전한 스마트 계약 기반 구매:
1. 미술품 세부 정보 검토
2. 검증 상태 확인
3. 구매 시작
4. 에스크로로 결제 보관
5. 추적 가능한 미술품 배송
6. 수령 확인
7. 결제 릴리스
8. 블록체인에서 소유권 이전

## 키보드 단축키

데스크톱용 파워 사용자 기능:

### 내비게이션
- `G` 다음 `H` - 홈으로 이동
- `G` 다음 `N` - 알림으로 이동
- `G` 다음 `P` - 프로필로 이동

### 작업
- `N` - 새 게시물(입력 중이 아닐 때)
- `?` - 모든 단축키 표시
- `Alt + T` - 알림 토글
- `/` - 검색에 포커스(출시 예정)

### 접근성
- `Tab` - 포커스 가능한 요소 탐색
- `Enter` - 포커스된 요소 활성화
- `Esc` - 모달 및 대화 상자 닫기
