# 계정 만들기

환영합니다! 이 가이드는 Ratel 계정을 만들고 숙의에 참여하거나 Fine-Art (d.AGIT) 마켓플레이스를 탐색할 준비를 하는 과정을 안내합니다.

## 시작하기 전에

다음을 준비하세요:
- 유효한 이메일 주소 또는 Google 계정
- 최신 웹 브라우저(Chrome, Firefox, Safari 또는 Edge)
- 가입 절차를 완료하기 위한 몇 분

## 가입 방법

Ratel은 계정을 만드는 두 가지 편리한 방법을 제공합니다:

![로그인 페이지](/img/sign-in-initial.png)

### 옵션 1: 이메일 가입

전통적인 이메일 기반 등록:

![이메일 가입 양식](/img/email-signup-form.png)

1. **Ratel 방문**
   - [https://dev.ratel.foundation](https://dev.ratel.foundation)으로 이동
   - 오른쪽 상단의 **로그인** 클릭

2. **등록 시작**
   - 로그인 페이지에서 **계정 만들기** 클릭
   - 이메일 주소 입력
   - **인증 코드 보내기** 클릭

3. **이메일 인증**
   - 받은 편지함에서 인증 코드 확인
   - 인증 필드에 6자리 코드 입력
   - 이메일이 보이지 않으면 스팸 폴더 확인
   - **인증** 클릭

4. **비밀번호 설정**
   - 강력한 비밀번호 생성(최소 8자)
   - 비밀번호를 다시 입력하여 확인
   - **계속** 클릭

5. **프로필 완성**
   - **표시 이름** 입력(다른 사람들이 보는 이름)
   - 고유한 **사용자 이름** 선택(예: @yourname)
   - 사용자 이름은 고유해야 하며 문자, 숫자, 밑줄을 포함할 수 있습니다
   - **가입 완료** 클릭

### 옵션 2: Google 가입

Google 계정을 사용한 빠른 가입:

위의 로그인 페이지에서 "Continue With Google" 버튼을 확인할 수 있습니다.

1. **Ratel 방문**
   - [https://dev.ratel.foundation](https://dev.ratel.foundation)으로 이동
   - 오른쪽 상단의 **로그인** 클릭

2. **Google 가입 선택**
   - **Google로 계속하기** 클릭
   - Google 인증 페이지로 리디렉션됩니다

3. **Google로 인증**
   - Google 계정 선택
   - Ratel이 요청하는 권한 검토
   - **허용** 클릭하여 액세스 승인

4. **프로필 완성**
   - 이메일이 자동으로 인증됩니다
   - **표시 이름**과 **사용자 이름** 입력
   - **가입 완료** 클릭

## 계정 생성 후

계정이 생성되면:

### 1. 환영 이메일

다음 내용이 포함된 환영 이메일을 받게 됩니다:
- 등록 확인
- 시작하기 리소스 링크
- 플랫폼에서의 첫 단계를 위한 팁

### 2. 프로필 설정

프로필 완성을 고려하세요:
- **프로필 사진**: 사진 또는 아바타 업로드(선택사항이지만 권장)
- **바이오**: 자신에 대한 간단한 설명 작성
- **관심사**: 관심 있는 주제 추가
- **자격 증명**: 검증 가능한 자격 증명 연결(선택사항)

자세한 프로필 사용자 정의는 [프로필 설정](../guide/account/profile-settings)을 참조하세요.

### 3. 플랫폼 탐색

이제 다음을 할 준비가 되었습니다:
- 피드에서 숙의 스페이스 탐색
- Fine-Art (d.AGIT) 마켓플레이스 탐색(출시 예정)
- 첫 게시물 작성
- 커뮤니티 토론 참여

## 계정 보안

### 비밀번호 모범 사례

이메일로 가입한 경우:
- 다른 곳에서 사용하지 않는 고유한 비밀번호 사용
- 최소 8자 권장
- 대문자, 소문자, 숫자, 기호 혼합
- 비밀번호 관리자 사용 고려

### 2단계 인증(출시 예정)

계정을 보호하기 위한 2FA를 통한 향상된 보안이 곧 제공될 예정입니다.

## 문제 해결

### 이메일 인증 문제

**인증 코드를 받지 못했나요?**
- 스팸/정크 폴더 확인
- 올바른 이메일 주소를 입력했는지 확인
- 몇 분 기다린 후 다시 시도
- 가능한 경우 "코드 재전송" 클릭

**인증 코드가 만료되었나요?**
- 코드는 10분 동안 유효합니다
- 계속하려면 새 코드를 요청하세요

### 사용자 이름이 이미 사용 중

원하는 사용자 이름을 사용할 수 없는 경우:
- 숫자 또는 밑줄 추가 시도(예: @yourname_1)
- 이름의 변형 사용
- 고유한 식별자로 창의적으로 작성

### Google 가입 문제

**Google과 연결할 수 없나요?**
- 브라우저에서 팝업이 활성화되어 있는지 확인
- Google 계정에 로그인되어 있는지 확인
- 다른 브라우저 사용 시도
- 브라우저 캐시를 지우고 다시 시도

## 개인정보 및 데이터

귀하의 개인정보는 저희에게 중요합니다:
- 계정 생성에 필요한 정보만 수집합니다
- 귀하의 이메일은 제3자와 절대 공유되지 않습니다
- 프로필 가시성을 제어할 수 있습니다
- 모든 데이터 처리는 업계 표준을 준수합니다

## 계정 유형

### 표준 계정(무료)

모든 사용자는 다음을 포함하는 표준 계정으로 시작합니다:
- 무제한 숙의 스페이스 참여
- 게시물 작성 및 커뮤니티 참여
- Fine-Art (d.AGIT) 마켓플레이스 액세스
- 기본 분석 및 인사이트

### 프리미엄 기능(출시 예정)

향후 멤버십 등급은 다음을 제공할 예정입니다:
- 스페이스 생성자를 위한 고급 분석
- 우선 지원
- 독점 기능 및 조기 액세스
- 향상된 사용자 정의 옵션

자세한 내용은 [멤버십](https://dev.ratel.foundation/membership)을 참조하세요.
