# 계정 문제

일반적인 계정 문제에 대한 해결 방법입니다.

## 로그인할 수 없음

### 비밀번호 재설정
1. 로그인 페이지에서 "비밀번호 찾기" 클릭
2. 이메일 입력
3. 이메일에서 재설정 링크 확인
4. 새 비밀번호 생성

### 이메일을 받지 못함
- 스팸/정크 폴더 확인
- noreply@ratel.foundation을 연락처에 추가
- 10분 기다린 후 다시 시도

## 계정 복구

### 이메일 액세스 상실
[hi@biyard.co](mailto:hi@biyard.co)로 다음 정보와 함께 이메일 보내기:
- 사용자 이름
- 대체 연락처 정보
- 신원 증명

### 계정 침해
1. 즉시 비밀번호 변경
2. 활성 세션 검토
3. 2FA 활성화
4. 필요시 지원팀 연락

## 계정 삭제

1. 설정 → **계정** → **계정 삭제**
2. 삭제 확인
3. 30일 유예 기간
4. 30일 후 영구 삭제
