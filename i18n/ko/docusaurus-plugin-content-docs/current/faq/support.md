# 지원 받기

필요할 때 도움을 받는 방법입니다.

- **문서**: 여기 있습니다!
- **FAQ**: 일반적인 질문
- **커뮤니티**: 스페이스에서 질문하기
- **상태 페이지**: ratel.foundation/status

## 지원팀 연락

### 이메일 지원
**이메일**: [hi@biyard.co](mailto:hi@biyard.co)

**응답 시간**:
- 일반: 48시간 이내
- 계정 문제: 24시간 이내
- 보안: 12시간 이내

**포함할 내용**:
- 문제에 대한 명확한 설명
- 스크린샷(관련 시)
- 사용자 이름
- 브라우저 및 OS
