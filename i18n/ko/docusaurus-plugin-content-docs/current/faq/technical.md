# 기술 질문

기술 문제 해결 및 요구사항입니다.

## 시스템 요구사항

### 브라우저 지원
- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+

### 인터넷 연결
- 최소: 1 Mbps
- 권장: 5+ Mbps

## 일반적인 문제

### 페이지가 로드되지 않음
- 브라우저 캐시 지우기
- 브라우저 확장 프로그램 비활성화
- 시크릿 모드 시도
- 인터넷 연결 확인

### 이미지를 업로드할 수 없음
- 파일 크기 확인(최대 10MB)
- 지원 형식: JPG, PNG, GIF, WebP
- 다른 브라우저 시도
- 인터넷 속도 확인

### 지갑이 연결되지 않음
- MetaMask 업데이트
- 지원되는 네트워크로 전환
- 지갑 캐시 지우기
- 다른 지갑 시도

## 성능

### 로딩 속도 느림
- 사용하지 않는 탭 닫기
- 브라우저 캐시 지우기
- 무거운 확장 프로그램 비활성화
- 인터넷 속도 확인
